# 수정 1: B-Roll 결과 저장 노드 추가

## 왜 필요한가

`B-Roll 결과 조회` 노드는 SplitInBatches 루프 안에서 5번 실행된다. 그런데 루프가 끝나고 밖에서 `$('B-Roll 결과 조회').all()`을 호출하면 **마지막 1개만** 나온다. n8n이 루프 안 노드의 데이터를 매번 덮어씌우기 때문이다.

그래서 루프 안에서 결과가 나올 때마다 `$getWorkflowStaticData('global')`에 **누적 저장**해야 한다.

## 현재 연결 구조

```
IF → (TRUE) → B-Roll 순차 생성 루프
```

## 변경할 연결 구조

```
IF → (TRUE) → 【B-Roll 결과 저장】 → B-Roll 순차 생성 루프
```

## 구체적 작업 순서

### 1-1. 노드 추가

1. 캔버스 빈 공간에서 **+** 버튼 클릭
2. 검색창에 **Code** 입력
3. **Code** 노드 클릭하여 추가
4. 노드 이름을 `B-Roll 결과 저장` 으로 변경
5. Mode: **Run Once for Each Item** (기본값)
6. Language: **JavaScript**

### 1-2. 코드 입력

아래 코드를 Code 필드에 붙여넣기:

```javascript
const staticData = $getWorkflowStaticData('global');

if (!staticData.brollResults) {
  staticData.brollResults = [];
}

const currentData = $input.first().json;
staticData.brollResults.push(currentData);

return [{
  json: {
    saved_index: staticData.brollResults.length,
    message: 'B-Roll 결과 ' + staticData.brollResults.length + '번째 저장 완료',
    data: currentData
  }
}];
```

### 1-3. 연결 변경

1. **IF 노드의 TRUE 출력** → `B-Roll 순차 생성 루프` 로 가는 **기존 선을 삭제**한다 (선 위에서 마우스 올려서 X 클릭)
2. **IF 노드의 TRUE 출력** → `B-Roll 결과 저장` 입력으로 연결
3. **B-Roll 결과 저장** 출력 → `B-Roll 순차 생성 루프` 입력으로 연결

# 수정 2: 토킹 영상 결과 저장 노드 추가

## 왜 필요한가

토킹 영상도 `토킹 클립 순차 생성 루프` (SplitInBatches) 안에서 5번 실행된다. B-Roll과 똑같은 이유로, 루프 밖에서 `$('토킹 영상 결과 조회').all()`을 호출하면 **마지막 1개만** 나온다.

## 현재 연결 구조

```
토킹 영상 결과 조회 → 토킹 클립 순차 생성 루프 (루프 반복)
```

## 변경할 연결 구조

```
토킹 영상 결과 조회 → 【토킹 영상 결과 저장】 → 토킹 클립 순차 생성 루프 (루프 반복)
```

## 구체적 작업 순서

### 2-1. 노드 추가

1. 캔버스 빈 공간에서 **+** 버튼 클릭
2. 검색창에 **Code** 입력
3. **Code** 노드 클릭하여 추가
4. 노드 이름을 `토킹 영상 결과 저장` 으로 변경
5. Mode: **Run Once for Each Item** (기본값)
6. Language: **JavaScript**

### 2-2. 코드 입력

아래 코드를 Code 필드에 붙여넣기:

```javascript
const staticData = $getWorkflowStaticData('global');

if (!staticData.talkingResults) {
  staticData.talkingResults = [];
}

const currentData = $input.first().json;
staticData.talkingResults.push(currentData);

return [{
  json: {
    saved_index: staticData.talkingResults.length,
    message: '토킹 영상 결과 ' + staticData.talkingResults.length + '번째 저장 완료',
```

### 2-3. 연결 변경

1. **토킹 영상 결과 조회** → `토킹 클립 순차 생성 루프` 로 가는 **기존 선을 삭제**한다
2. **토킹 영상 결과 조회** 출력 → `토킹 영상 결과 저장` 입력으로 연결
3. **토킹 영상 결과 저장** 출력 → `토킹 클립 순차 생성 루프` 입력으로 연결

# 수정 3: 최종 결과 집계 — 코드 전체 교체

`최종 결과 집계` 노드를 더블클릭 → 코드 영역 클릭 → **Ctrl+A** (전체 선택) → 아래 코드를 **Ctrl+V** (붙여넣기):

```javascript
// ============================================
// 최종 결과 집계 — v7.2 최종 확정 (2026.03.05)
// B-Roll + 토킹 모두 Static Data에서 가져오기
// ============================================

// 1) 디렉터 AI 원본 5개 item
const directorItems = $('Code in JavaScript1').all();

// 2) 대본 파싱에서 브랜드/제품 정보
const scriptData = $('대본 파싱').first().json;

// 3) ★ Static Data에서 B-Roll + 토킹 결과 모두 가져오기
const staticData = $getWorkflowStaticData('global');
const brollResults = staticData.brollResults || [];
const talkingResults = staticData.talkingResults || [];

// ── URL 추출 함수 ──
function extractVideoUrl(pollData) {
  if (!pollData) return '';
  const d = pollData.json || pollData;
  try {
    // 경로 A: d.data.response.resultUrls[0]
    if (d.data && d.data.response && d.data.response.resultUrls && d.data.response.resultUrls[0]) {
      return d.data.response.resultUrls[0];
    }
    // 경로 B: d.response.resultUrls[0]
    if (d.response && d.response.resultUrls && d.response.resultUrls[0]) {
      return d.response.resultUrls[0];
    }
    // 경로 C: d.data.response.resultUris[0]
    if (d.data && d.data.response && d.data.response.resultUris && d.data.response.resultUris[0]) {
      return d.data.response.resultUris[0];
    }
    // 경로 D: d.response.resultUris[0]
    if (d.response && d.response.resultUris && d.response.resultUris[0]) {
      return d.response.resultUris[0];
    }
    // 경로 E: resultJson 파싱
    const rj = d.data?.resultJson || d.resultJson || '';
    if (rj) {
      const parsed = typeof rj === 'string' ? JSON.parse(rj) : rj;
      return parsed.video?.url || parsed.url || parsed.output?.[0] || '';
    }
  } catch(e) {
    return '';
  }
  return '';
}

// ── 토킹 클립 5개 조립 (Static Data에서 가져온 talkingResults 사용) ──
const talkingClips = [];
for (let i = 0; i < directorItems.length; i++) {
  const dir = directorItems[i].json;
  const videoUrl = extractVideoUrl(talkingResults[i]);
  talkingClips.push({
    segment_id: dir.timeline_id,
    section: dir.section,
    time_range: dir.time_range,
    tts_text: dir.tts_text,
    video_url: videoUrl,
    status: videoUrl ? 'completed' : 'missing'
  });
}

// ── B-Roll 클립 5개 조립 (Static Data에서 가져온 brollResults 사용) ──
const brollClips = [];
for (let i = 0; i < directorItems.length; i++) {
  const dir = directorItems[i].json;
  const videoUrl = extractVideoUrl(brollResults[i]);
  brollClips.push({
    broll_id: 'broll_' + (i + 1),
    broll_name: dir.broll_name || ('B-Roll ' + (i + 1)),
    section: dir.section,
    time_range: dir.time_range,
    video_url: videoUrl,
    status: videoUrl ? 'completed' : 'missing'
  });
}

// ── 집계 ──
const talkingOk = talkingClips.filter(c => c.status === 'completed').length;
const brollOk = brollClips.filter(c => c.status === 'completed').length;

// ★ Static Data 초기화 (다음 워크플로우 실행을 위해)
staticData.brollResults = [];
staticData.talkingResults = [];

return [{
  json: {
    brand: scriptData.brand || '',
    product: scriptData.product || '',
    price: scriptData.price || '',
    full_script: scriptData.script?.full_script || directorItems[0]?.json.full_script || '',
    talking_clips: talkingClips,
    broll_clips: brollClips,
    totals: {
      talking: talkingOk + '/5',
      broll: brollOk + '/5'
    },
    status: (talkingOk === 5 && brollOk === 5) ? 'ALL_COMPLETE' : 'PARTIAL',
    next_step: (talkingOk === 5 && brollOk === 5)
      ? '전체 완료. 영상 편집 진행 가능.'
      : 'URL 누락 항목 있음. missing 항목 확인 후 재실행.'
  }
}];
```

# 수정 4: B-Roll 프롬프트 5개 생성 — 코드 전체 교체

`B-Roll 프롬프트 5개 생성` 노드를 더블클릭 → 코드 영역 클릭 → **Ctrl+A** (전체 선택) → 아래 코드를 **Ctrl+V** (붙여넣기):

```javascript
// ============================================
// B-Roll 프롬프트 5개 생성 — v7.2 최종 확정 (2026.03.05)
// Section 기반 generation_type + 극사실주의 프롬프트 강화
// ============================================

const directorItems = $('Code in JavaScript1').all();
const scriptData = $('대본 파싱').first().json;

const productImageMain = scriptData.product_image_main || '';

// ★ Discovery와 Solution만 IMAGE_2_VIDEO (제품 이미지 기반)
// ★ Hook, Problem, CTA는 TEXT_2_VIDEO (텍스트만으로 영상 생성)
const IMAGE_SECTIONS = ['discovery', 'solution'];

// ★ 극사실주의 영상 프롬프트 접미사 (인물/라이프스타일 씬용)
const HYPERREAL_VIDEO_SUFFIX = 'hyper-realistic footage indistinguishable from real cinema camera capture, ultra-detailed skin with visible pores and micro expressions and natural skin texture movement during motion, realistic hair physics with individual strands responding to movement and gravity and breeze naturally, natural subtle body micro-movements like breathing pulse and muscle tension shifts and weight distribution changes, realistic eye movement with natural blink rate and eye moisture and pupil dilation, fabric physics responding realistically to body movement and air and gravity, natural motion blur consistent with real camera shutter speed, absolutely ZERO artificial smooth plastic skin, ZERO uncanny valley, ZERO AI-generated look, ZERO robotic stiff movement, must pass as genuine footage from professional cinema camera';

// ★ 제품 전용 프롬프트 접미사 (제품 씬용)
const HYPERREAL_PRODUCT_SUFFIX = 'hyper-realistic product photography indistinguishable from real studio photograph, ultra-detailed material textures with visible surface micro-texture and natural light interaction, realistic glass or plastic refraction and reflection with visible internal light caustics, natural shadow falloff with realistic ambient occlusion and contact shadows, visible label printing texture and slight label edge lifting, realistic liquid meniscus visible through transparent container, shot on medium format camera with natural lens characteristics and shallow depth of field, absolutely ZERO CGI render look, ZERO 3D modeling aesthetic, must pass as genuine product photo from commercial studio shoot';

// ★ section별 시네마틱 프롬프트 프리픽스
const SECTION_PROMPT_PREFIX = {
  hook: 'Cinematic 9:16 vertical video, Korean woman in her late 20s doing morning skincare routine in bright natural-lit modern Korean bathroom, warm golden morning light streaming through frosted window, soft steam from recent shower visible in air, she looks into mirror touching her face examining skin, wearing cozy oversized white cotton t-shirt, shot on Sony FX3 with Sony FE 35mm f/1.4 GM at ISO 800 3200K warm tone, shallow depth of field with bokeh on bathroom background, authentic candid moment captured mid-action, ',
  problem: 'Cinematic 9:16 vertical close-up video, extreme close-up of dry flaky skin on forearm and neck area under harsh fluorescent bathroom light revealing texture and dryness, visible dry patches and rough texture, the person gently touches the dry area showing discomfort, natural skin with visible pores and imperfections, shot on Sony A7R V with Sony FE 90mm f/2.8 Macro at ISO 400 5000K neutral daylight, macro-level skin detail visible, ',
  discovery: 'Cinematic 9:16 vertical video, elegant product hero shot with the body mist bottle center frame on marble bathroom counter, soft diffused natural window light creating beautiful caustics through glass bottle, gentle mist spray being released from bottle nozzle in slow motion with visible fine droplets catching light, clean minimalist Korean bathroom aesthetic background, shot on Canon EOS R5 with Canon RF 85mm f/1.2L at ISO 200 4500K, ',
  solution: 'Cinematic 9:16 vertical video, close-up of Korean woman spraying body mist on her wrist and inner arm in slow motion, fine mist particles visible in golden hour sunlight streaming sideways, she gently rubs wrists together then brings to nose with satisfied subtle smile, visible skin moisture and glow after application, soft warm bedroom background with linen curtains, shot on Sony FX3 with Sony FE 50mm f/1.2 GM at ISO 640 3500K warm tone, ',
  cta: 'Cinematic 9:16 vertical video, medium shot of Korean woman in her late 20s holding the product near her face with genuine satisfied smile, warm afternoon golden hour light from window creating rim lighting on her hair, she looks directly at camera with confident happy expression then gives subtle nod of approval, cozy modern Korean apartment living room background with plants, shot on Sony A7IV with Sony FE 35mm f/1.4 GM at ISO 800 4000K, '
};

const brolls = directorItems.map((item, i) => {
  const dir = item.json;
  const section = (dir.section || '').toLowerCase();
  const isProductScene = IMAGE_SECTIONS.includes(section);

  // 원래 broll_prompt에 프리픽스와 접미사를 결합
  const originalPrompt = dir.broll_prompt || '';
  const prefix = SECTION_PROMPT_PREFIX[section] || '';
  const suffix = isProductScene ? HYPERREAL_PRODUCT_SUFFIX : HYPERREAL_VIDEO_SUFFIX;
  const enhancedPrompt = prefix + originalPrompt + ', ' + suffix;

  // veo_body 조립
  let veoBody;
  if (isProductScene && productImageMain) {
    veoBody = {
      prompt: enhancedPrompt,
      model: 'veo3_fast',
      generationType: 'IMAGE_2_VIDEO',
      aspect_ratio: '9:16',
      image_url: productImageMain,
      enableTranslation: false
    };
  } else {
    veoBody = {
      prompt: enhancedPrompt,
      model: 'veo3_fast',
      generationType: 'TEXT_2_VIDEO',
      aspect_ratio: '9:16',
      enableTranslation: false
    };
  }

  return {
    json: {
      id: 'broll_' + (i + 1),
      name: dir.broll_name || ('B-Roll ' + (i + 1)),
      section: dir.section,
      time_range: dir.time_range,
      prompt: enhancedPrompt,
      generation_type: isProductScene ? 'IMAGE_2_VIDEO' : 'TEXT_2_VIDEO',
      image_url: isProductScene ? productImageMain : '',
      product_image_url: productImageMain,
      veo_body: JSON.stringify(veoBody),
      brand: dir.brand || scriptData.brand || '',
      product: dir.product || scriptData.product || '',
      full_script: dir.full_script || ''
    }
  };
});

return brolls;
```
