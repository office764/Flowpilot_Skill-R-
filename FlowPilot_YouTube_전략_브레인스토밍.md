## 3. 리드 생성 퍼널 설계

### 퍼널 구조: 영상 → 무료 가치 → 리드 수집 → 세일즈 콜

```
[YouTube 영상 시청]
        │
        ▼
[영상 내 CTA: "무료 자동화 진단 받기"]
        │
        ▼
[랜딩페이지: flowpilots.org/free-audit]
   - 회사명
   - 담당자 이름
   - 이메일
   - 전화번호
   - "가장 시간 많이 쓰는 반복업무는?" (드롭다운)
   - "월 몇 시간 소요?" (슬라이더)
        │
        ▼
[자동 발송: 맞춤 자동화 진단 보고서 PDF]
   (n8n 워크플로우로 자동 생성)
        │
        ▼
[3일 후 팔로업 이메일: "진단 결과 확인하셨나요? 15분 무료 상담 예약하세요"]
        │
        ▼
[세일즈 콜 (15분)]
   - 고충 심층 파악
   - ROI 수치 제시
   - 프로젝트 제안
        │
        ▼
[제안서 발송 → 계약 → 온보딩]
```

### 각 영상에 심을 CTA 스크립트

**오프닝 CTA (영상 시작 30초):**
> "이 영상 끝까지 보시면 여러분 회사에 바로 적용할 수 있는 자동화 팁을 드릴 텐데요,
> 만약 '우리 회사는 정확히 뭘 자동화해야 하지?' 궁금하시면
> 설명란 링크에서 무료 자동화 진단을 신청해 보세요."

**미드 CTA (영상 중간):**
> "지금 보여드린 이 워크플로우, 실제로 우리 고객사에 적용해서
> 월 40시간 절감한 케이스입니다.
> 여러분 회사에도 적용 가능한지 궁금하시면 무료 진단 신청 링크 확인해 보세요."

**엔딩 CTA (영상 마지막):**
> "오늘 영상이 도움이 되셨다면 구독과 좋아요 부탁드리고요,
> 설명란에 무료 자동화 진단 링크 남겨놨습니다.
> 여러분 회사의 반복업무를 분석해서 맞춤 자동화 로드맵을 무료로 만들어 드립니다.
> 다음 영상에서 만나요!"

### 리드 자동화 워크플로우 (n8n)

리드가 들어오면 자동으로 처리하는 n8n 워크플로우:

```
1. Webhook 노드 (랜딩페이지 폼 제출 수신)
2. Google Sheets 노드 → Operation: "Append Row" (리드 DB에 기록)
3. OpenAI 노드 → Operation: "Message a Model" (고객 업무 기반 맞춤 진단 보고서 생성)
4. HTML to PDF 변환 (Code 노드)
5. Gmail 노드 → Operation: "Send" (진단 보고서 PDF 첨부 발송)
6. Wait 노드 → 3일 대기
7. Gmail 노드 → Operation: "Send" (팔로업 이메일: 세일즈 콜 예약 링크)
8. Slack 노드 → Operation: "Send a Message" (내부 알림: "새 리드 인입!")
```

## 10. CMO 인사이트: 왜 YouTube인가?

### 핵심 논리

1. **검색 엔진 2위**: 유튜브는 구글 다음으로 큰 검색 엔진. "n8n 사용법", "업무 자동화" 검색하는 잠재고객이 바로 우리 타겟
2. **롱테일 리드 머신**: 한 번 올린 영상이 1년, 2년 뒤에도 리드를 계속 가져옴. 광고비 0원으로 유입되는 에버그린 자산
3. **신뢰 구축 최강**: 10분짜리 영상 하나가 영업사원의 1시간 설명보다 강력함
4. **세일즈 콜 워밍업**: 세일즈 콜 전에 "이 영상 3개만 보세요"로 사전 교육 → 콜에서 바로 클로징
5. **CPL(리드 획득 단가) 최저**: 유료 광고 CPL 대비 유튜브 오가닉 리드의 CPL이 압도적으로 낮음

### 수치 예측 (3개월 후)

- 유튜브 오가닉 리드 CPL: 약 0원 (콘텐츠 제작 인건비만 소요)
- 메타 광고 리드 CPL: 약 15,000~30,000원
- 유튜브 리드의 세일즈 콜 전환율: 30~40% (이미 영상으로 교육된 상태)
- 메타 광고 리드의 세일즈 콜 전환율: 10~15%
