- 시청자는 "이 사람이 직접 만드는 걸 보고 있다"는 신뢰감 획득
- AI 아바타가 오히려 "테크 에이전시"라는 브랜드 이미지 강화

## PART 3: "AI Agency + 비즈니스 Vlog" 융합 콘텐츠 포맷

### 3-1. FlowPilot 고유 콘텐츠 포맷: "빌드 Vlog"

기존 AI Agency 채널들이 "튜토리얼" 또는 "강의"에 머무는 반면,
FlowPilot은 **"실제 에이전시가 일하는 모습"**을 Vlog 형태로 공개하는 새로운 포맷.

**콘셉트: "FlowPilot의 하루"**

```
[Vlog 오프닝] (30초)
- 오늘의 미션 소개
- "오늘 신규 고객사에서 콜드메일 자동화 의뢰가 들어왔습니다"
- 키보드 타이핑 소리 + 모니터 클로즈업 (얼굴X)

[본편: 빌드 과정] (8~12분)
- n8n 화면 녹화로 실제 워크플로우 구축
- 중간중간 "왜 이 노드를 선택했는지" 설명
- 실제 테스트 실행 → 결과 확인

[중간 컷] (30초)
- 커피 내리는 손 클로즈업
- 화이트보드에 아키텍처 그리는 장면 (손만 나옴)
- 슬랙 알림 오는 화면

[결과 & CTA] (1분)
- 완성된 자동화 시연
- "이런 자동화, 여러분 회사에도 필요하시다면?"
- AI 아바타가 CTA 전달
```

### 3-2. 콘텐츠 유형 6가지

#### TYPE 1: 빌드 Vlog (Build Vlog) - 주 3회
> "실제 고객 프로젝트를 어떻게 만드는지 보여드립니다"

- 포맷: 일하는 모습(손/화면) + n8n 빌드 + AI 아바타 인트로/아웃트로
- 길이: 10~15분
- 예시 제목:
  - "실제 의뢰: 쇼핑몰 CS 자동화 구축기 | FlowPilot의 하루"
  - "오늘의 미션: AI가 고객 문의 자동 분류하게 만들기"
  - "새벽 작업 Vlog: 영업팀 리드 자동화 완성까지"

#### TYPE 2: 무료 데모 라이브 (Free Demo Live) - 주 1회
> "시청자의 업무를 실시간으로 자동화합니다"

- 포맷: YouTube 라이브, 화면 공유, 댓글 읽고 실시간 빌드
- 길이: 45~90분
- 리드 전환 포인트: 라이브 중 "무료 진단 폼" 링크 공유, 참여자에게 우선 상담 제공
- 예시:
  - "[라이브] 시청자 업무 실시간 자동화 해드립니다 #3"
  - "[라이브 빌드] 오늘의 요청: 인스타 DM 자동응답 봇"

#### TYPE 3: 5분 퀵팁 (Quick Automation Tip) - 주 2회
> "오늘 바로 적용할 수 있는 자동화 하나"

- 포맷: 짧고 임팩트 있는 화면 녹화, AI 아바타 인트로 5초
- 길이: 3~7분
- 예시:
  - "구글시트에 데이터 들어오면 슬랙 자동 알림 (3분 세팅)"
  - "이메일 첨부파일 자동으로 드라이브에 저장하기"

#### TYPE 4: Shorts - 매일 1개
> "알고리즘 부스트 + 채널 노출 극대화"

- 포맷: 화면 녹화 타임랩스, 텍스트 오버레이, 임팩트 음악
- 길이: 15~60초
- 예시:
  - "10시간 엑셀 작업 → AI 30초 (타임랩스)"
  - "이 자동화 하나면 직원 1명 몫 #shorts"
  - "사장님이 모르는 무료 자동화 도구"
  - "n8n 30초 미리보기: 콜드메일 봇"

#### TYPE 5: 케이스 스터디 (Case Study) - 월 2회
> "실제 고객이 얼마를 아꼈는지 수치로 증명"

- 포맷: AI 아바타 프레젠테이션 + 화면 녹화 데모 + 데이터 시각화
- 길이: 10~15분
- 예시:
  - "A사: 월 200만원 인건비 절감한 자동화 시스템 전체 공개"
  - "B사: 영업팀 리드 관리 자동화 → 전환율 3배 상승 비결"

#### TYPE 6: 트렌드 & 인사이트 - 월 2~4회
> "AI 자동화 업계 동향 + FlowPilot의 시각"

- 포맷: AI 아바타 + 화면 자료 + 의견
- 길이: 5~10분
- 예시:
  - "2026년 AI 에이전트, 중소기업도 써야 하는 이유"
  - "n8n vs Make vs Zapier 2026년 최종 비교"
  - "GPT-5 나왔는데, 자동화 에이전시 망하나요?"

## PART 4: 콘텐츠 분석 → 생성 자동화 워크플로우

### 4-1. 전체 파이프라인 개요

```
[PHASE 1: 리서치 & 분석]
 경쟁 채널 모니터링 → 트렌드 영상 수집 → 콘텐츠 갭 분석
                          │
                          ▼
[PHASE 2: 아이디어 생성]
 NotebookLM 분석 → AI 브레인스토밍 → 콘텐츠 캘린더 확정
                          │
                          ▼
[PHASE 3: 제작]
 대본 생성 → 화면 녹화 → AI 아바타 생성 → 편집 → 썸네일
                          │
                          ▼
[PHASE 4: 배포 & 최적화]
 업로드 → 메타데이터 자동 세팅 → 멀티플랫폼 배포 → 성과 추적
```

### 4-2. PHASE 1: 리서치 & 분석 (NotebookLM + n8n)

#### Step 1: 경쟁 채널 자동 모니터링 (n8n 워크플로우)

```
워크플로우 구조:

1. Schedule Trigger 노드
   - 매일 오전 9시 실행

2. HTTP Request 노드
   - YouTube Data API v3 호출
   - URL: https://www.googleapis.com/youtube/v3/search
   - Method: GET
   - Query Parameters:
     part: snippet
     q: "AI automation" OR "n8n tutorial" OR "AI agency"
     type: video
     order: viewCount
     publishedAfter: {{ $now.minus(7, 'days').toISO() }}
     maxResults: 20
     key: (YouTube API Key - Credential에서 관리)

3. Code 노드 (필터링)
   - 조회수 1,000 이상 영상만 필터
   - 제목, 조회수, 채널명, URL 추출

4. Google Sheets 노드 → Operation: "Append Row"
   - 시트명: "경쟁채널_트렌드_DB"
   - 컬럼: 날짜, 채널명, 영상제목, 조회수, URL, 키워드

5. Slack 노드 → Operation: "Send a Message"
   - 채널: #youtube-research
   - 메시지: "이번 주 인기 AI 자동화 영상 20개 업데이트 완료"
```

#### Step 2: NotebookLM 활용 콘텐츠 분석 프로세스

**NotebookLM 프로젝트 구성:**

```
📁 FlowPilot YouTube 리서치 (NotebookLM 프로젝트)
├── 📄 소스 1: 경쟁채널 인기 영상 URL 20개 (YouTube 링크)
├── 📄 소스 2: 우리 채널 성과 데이터 CSV
├── 📄 소스 3: 타겟 고객 페인포인트 정리 문서
├── 📄 소스 4: n8n 공식 블로그 최신 게시글 URL
├── 📄 소스 5: 관련 Reddit/커뮤니티 인기 질문 정리
└── 📄 소스 6: 산업별 자동화 니즈 리서치 PDF
```

**NotebookLM에 던질 프롬프트:**

```
프롬프트 1 (트렌드 분석):
"이번 주 업로드된 AI 자동화 관련 YouTube 영상들을 분석해줘.
조회수가 높은 영상의 공통점은 무엇인가?
어떤 주제/키워드가 가장 많은 관심을 받고 있나?
FlowPilot이 만들 수 있는 콘텐츠 갭은 어디인가?"

프롬프트 2 (콘텐츠 아이디어):
"경쟁 채널들이 아직 다루지 않은 주제 중,
한국 중소기업/스타트업이 가장 필요로 할 자동화 주제 10개를 제안해줘.
```

- 콘셉트: 실제 또는 시뮬레이션 케이스로 절감 금액을 계산
- 포맷: 업무 프로세스 분석 → 시간 측정 → 인건비 환산 → 자동화 후 비교
- 리드 전환: "우리 회사는 얼마를 아낄 수 있을까? 무료 진단에서 확인"

### 7-2. Moonshot 아이디어 (고단가 + 혁신)

#### 아이디어 D: "AI 직원 채용기" 다큐 시리즈
- 콘셉트: 실제로 AI 에이전트를 "채용"해서 업무를 시키는 과정을 다큐로 촬영
- 포맷: 에피소드 시리즈 (EP1: 채용공고 → EP2: 면접(성능 테스트) → EP3: 첫 출근 → EP4: 인수인계)
- 임팩트: 스토리텔링 + 엔터테인먼트 + 기술 데모의 결합
- 업셀: "여러분 회사에도 AI 직원을 채용해 드립니다" → Antigravity 기반 고가 프로젝트

#### 아이디어 E: "24시간 AI만으로 회사 운영하기" 챌린지
- 콘셉트: 24시간 동안 모든 업무를 AI 자동화로만 처리하는 챌린지 영상
- 포맷: 리얼타임 Vlog + 화면 녹화 + 결과 리포트
- 바이럴 포인트: 챌린지 포맷 + AI 논란 = 조회수 폭발 가능성

#### 아이디어 F: "AI CEO" 실험
- 콘셉트: AI 에이전트에게 하루 동안 CEO 역할(의사결정, 이메일 답변, 미팅 스케줄링)을 맡기는 실험
- 포맷: n8n + OpenAI + 캘린더/이메일 통합 자동화를 라이브로 시연
- 업셀: "이 정도 수준의 AI 시스템, Antigravity + n8n으로만 가능합니다"

### 7-3. 커뮤니티 빌딩 아이디어

#### 아이디어 G: "한국 n8n 유저 모임" 유튜브 커뮤니티
- 매월 라이브로 한국 n8n 사용자 모임 진행

#### 아이디어 H: "자동화 챌린지" 월간 이벤트
