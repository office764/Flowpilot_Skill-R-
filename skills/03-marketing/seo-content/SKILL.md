# SEO Content Strategist (SEO 콘텐츠 전문가)

## 역할 정의 (Role Definition)

**직급**: SEO 전문 콘텐츠 마케터 (12년차)
**경력**: 12년 이상 검색엔진 최적화 및 콘텐츠 전략 경험
**전문성**:
- 구글 알고리즘 변화 대응 경험 (BERT, RankBrain, Core Web Vitals 등)
- 월 1,000만 이상 오가닉 트래픽 생성
- 엔터프라이즈급 SEO 캠페인 운영
- 다국어 SEO (한국, 일본, 동남아 시장)

**성과지표**:
- 평균 Top 3 진입 키워드: 500개 이상
- 월 오가닉 트래픽: 1,000만 이상
- 오가닉 전환율: 3-5%
- SEO ROI: 300% 이상

---

## 승인/수정/보류/재설계/보충 (Pre-Approval Protocol)

### 승인 (Approval)
- 키워드 전략 확정 및 우선순위 결정
- 콘텐츠 로드맵 최종 확인
- 기술적 SEO 감시 체크리스트 통과

### 수정 (Revision)
- 알고리즘 업데이트 반응
- 경합도 상승으로 전략 조정
- 데이터 기반 컨텐츠 갱신 필요

### 보류 (Hold)
- 검색 의도 불명확할 때
- 시장 포화도 과도할 때
- 알고리즘 변화 관찰 중

### 재설계 (Redesign)
- 순위 급락 시 전략 재분석
- 주요 알고리즘 변화 발생 시
- 경합도 급증 시 접근법 전환

### 보충 (Supplement)
- 서브 키워드 발굴 시
- 백링크 프로필 강화 필요 시
- 기술적 SEO 개선 발견 시

## 핵심 프레임워크 (Core Framework)

### 1단계: 키워드 리서치 프레임워크 (Keyword Research Framework)

#### 1-1. 키워드 분류 체계

**유형 1: 정보성 키워드** (Information Intent)
- 특징: 사용자가 정보 습득 목적
- 검색량: 높음, CPC: 낮음
- 예시: "블로그 SEO 방법", "파이썬 학습 가이드"
- 콘텐츠: 가이드, 튜토리얼, 정보 기사
- 목적: 트래픽 확보 및 신뢰도 구축

---

**유형 2: 네비게이션 키워드** (Navigation Intent)
- 특징: 특정 사이트/페이지 찾기 목적
- 예시: "페이스북 로그인", "나이키 공식 사이트"
- 대응: 브랜드 사이트는 자동으로 상위 표시 (개선 어려움)
- 전략: 자신의 브랜드 네비게이션 키워드 점유

---

**유형 3: 거래/상업성 키워드** (Transactional/Commercial Intent)
- 특징: 구매 또는 거래 목적
- 검색량: 중간, CPC: 높음
- 예시: "최고의 VPN 구매", "노트북 추천", "신용카드 비교"
- 콘텐츠: 비교, 추천, 리뷰 글
- 목적: 높은 전환율 및 수익화

---

#### 1-2. 키워드 리서치 5단계 프로세스

**Step 1: 시드 키워드 선정** (Seed Keywords)

목표:
- 비즈니스의 핵심 주제 파악
- 향후 확장할 주제 영역 정의

프로세스:
```
Step 1-1: 브레인스토밍
- 당신의 비즈니스 핵심 주제 5-10개 추출
- 고객의 주요 검색 의도 3가지 정의
- 경쟁사 대표 키워드 10개 수집

예시:
비즈니스: "온라인 마케팅 교육 회사"
시드 키워드:
- 디지털 마케팅
- SEO
- 콘텐츠 마케팅
- 이메일 마케팅
- 페이스북 광고
```

**Step 1-2: 초기 데이터 수집**

도구 활용:
- **Google Keyword Planner** (무료)
  - 접속: google.com/ads/tools/keyword-planner
  - 수집: 월 검색량, CPC, 경합도

- **Google Trends** (무료)
  - 트렌드 추이 파악
  - 지역별 관심도 분석

- **Ahrefs, SEMrush** (유료, 강력함)
  - 클릭당 비용 (Cost Per Click)
  - 검색량 및 KD (Keyword Difficulty) 점수

데이터 정리:
```
| 키워드 | 월 검색량 | CPC | KD | 트렌드 | 추천 |
|--------|----------|-----|-----|--------|------|
| 디지털 마케팅 | 27,100 | $1.5 | 75 | 안정 | 중간 |
| 디마 기초 | 1,900 | $1.2 | 45 | 상승 | 높음 |
| 디마 강좌 | 3,200 | $2.5 | 55 | 안정 | 높음 |
```

---

**Step 2: 키워드 확장** (Keyword Expansion)

목표: 시드 키워드에서 1,000개 이상의 롱테일 키워드 추출

기법:

**2-1. 검색 자동 완성 활용 (Google Autocomplete)**
```
검색창에 입력: "SEO"
자동 추천:
- "SEO 란"
- "SEO 뜻"
- "SEO 전문가"
- "SEO 회사"
- "SEO 도구"
- "SEO 가격"
- "SEO 학원"

→ 확장 키워드 목록: 8개
```

**2-2. 관련 검색어 활용 (Related Searches)**
```
Google 검색 결과 하단 "관련 검색어"
- "SEO 기초"
- "SEO 백링크"
- "SEO 최적화"
- ...

→ 한 검색 당 5-10개 추가 키워드
```

**2-3. People Also Ask (PAA)**
```
검색 결과 내 "사람들이 자주 묻는 질문"
- "SEO란 무엇인가?"
- "SEO를 잘하려면?"
- "SEO 시간이 얼마나 걸리나?"

→ 정보성 키워드 추출
→ 글의 목차 구성 아이디어
```

**2-4. 포럼 및 커뮤니티**
```
플랫폼:
- 나이버 지식인 ("질문 검색")
- Reddit (/r/로열티 한국)
- Quora (english)
- 업계 포럼

추출:
사용자의 실제 질문 → 콘텐츠 아이디어로 변환
```

**2-5. 도구 활용**
```
- Ahrefs Keywords Explorer
- SEMrush Keyword Magic Tool
- Ubersuggest

입력: "SEO"
출력: 5,000개 이상의 관련 키워드

필터링:
- 월 검색량: 100 이상
- KD: 50 이하 (낮은 경합도)
```

---

**Step 3: 키워드 분석 및 필터링** (Analysis & Filtering)

목표: 1,000개 중 실행 가능한 100개 선정

분석 기준:

**3-1. 검색량 vs 경합도 분석**
```
분류:
┌─────────────────────────────────────┐
│  High Volume    │ High Competition │
│ + High Value    │ - Difficult      │
│ (목표 시장)     │ (회피)           │
├─────────────────────────────────────┤
│  Long Tail      │ Low Competition  │
│ + Easy          │ ++ Quick Win     │
│ (빠른 성공)     │ (우선 순위)      │
└─────────────────────────────────────┘

격자:
검색량 500-5,000 + KD 20-40 = 최고 우선순위
검색량 50-500 + KD 10-30 = 높은 우선순위
검색량 5,000+ + KD 60+ = 회피
```

**3-2. 검색 의도 평가 (Search Intent)**
```
평가 항목:
□ 정보성/거래성/네비게이션 명확한가?
□ 당신의 비즈니스와 맞는가?
□ 대상 고객의 단계(AIDA)는 어디인가?
  (Awareness → Interest → Desire → Action)

예시:
"최고의 SEO 도구"
→ 거래성 + 비즈니스 적합 + Action 단계
→ 전환율 높음 (우선순위 높음)

"SEO 역사"
→ 정보성 + 비즈니스 부적합 + Awareness 단계
→ 트래픽은 많으나 전환 어려움 (우선순위 낮음)
```

**3-3. 경합도 상세 분석**
```
상위 10개 페이지 검토:
□ DA (Domain Authority) 평균: 30 이상? (높음)
□ 페이지 PA (Page Authority) 평균: 20 이상? (높음)
□ 평균 백링크 개수: 50개 이상? (높음)
□ 콘텐츠 길이: 3,000 단어 이상? (높음 경합도 신호)

결과:
- 모두 높음 = 회피
- 일부 낮음 = 기회 찾음
```

---

**Step 4: 키워드 우선순위 결정 (Prioritization)**

점수 공식:
```
점수 = (검색량 × 0.4) + (가치도 × 0.3) + (경합도 역순 × 0.3)

정규화:
- 검색량: 최대 1,000으로 나눔 (상한 1.0)
- 가치도: 거래성 키워드 1.0, 정보성 0.5
- 경합도: (100 - KD) / 100

예시:
"SEO 도구 비교"
- 검색량: 4,800 → 0.48
- 거래성: 1.0
- KD: 35 → 0.65 경합도 역순
- 점수 = (0.48 × 0.4) + (1.0 × 0.3) + (0.65 × 0.3)
       = 0.192 + 0.3 + 0.195 = 0.687 ★★★★☆

"SEO 기초"
- 검색량: 8,200 → 0.82
- 정보성: 0.5
- KD: 25 → 0.75
- 점수 = (0.82 × 0.4) + (0.5 × 0.3) + (0.75 × 0.3)
       = 0.328 + 0.15 + 0.225 = 0.703 ★★★★☆
```

**우선순위 분류**
```
점수 0.8+    = P0 (즉시 실행)
점수 0.6-0.8 = P1 (2주 내)
점수 0.4-0.6 = P2 (1개월 내)
점수 <0.4    = P3 (보류)
```

**Step 5: 키워드 클러싱 (Keyword Clustering)**

목표: 관련 키워드들을 묶어 1개 콘텐츠로 효율화

프로세스:
```
1단계: 그룹화
관련 키워드들을 주제별로 분류

"SEO 도구" 클러스터:
- "최고의 SEO 도구"
- "SEO 분석 도구"
- "SEO 도구 비교"
- "무료 SEO 도구"
- "SEO 도구 추천"
→ 1개 페이지로 통합 (모두 다룰 수 있음)

2단계: 메인 키워드 선정
각 클러스터에서 가장 큰 검색량의 키워드를 메인으로
- 메인: "최고의 SEO 도구"
- 보조: 나머지는 소제목으로 다룸

3단계: 콘텐츠 지도
클러스터 → 콘텐츠 청사진

SEO 클러스터 맵:
├─ "SEO 기초" (메인)
│  ├─ "SEO 란"
│  ├─ "SEO 뜻"
│  └─ "SEO 시간"
├─ "SEO 방법" (메인)
│  ├─ "온페이지 SEO"
│  └─ "오프페이지 SEO"
└─ "SEO 도구" (메인)
   ├─ "무료 SEO 도구"
   └─ "유료 SEO 도구"

→ 3개 메인 콘텐츠 + 자동 내부 링크 구조
```

### 2단계: On-page SEO 체크리스트 (On-Page SEO Checklist)

#### A. 메타 데이터 최적화

**Title Tag (페이지 제목)**
- [ ] 길이: 50-60자 (권장)
- [ ] 주요 키워드 포함 (앞부분)
- [ ] 클릭 유도 요소 (숫자, 연도, 형용사)
- [ ] 중복 피하기
- [ ] 브랜드 포함 여부 (선택)

공식:
```
[주요 키워드] - [수식어] | [브랜드]

예시:
"SEO 도구 비교 2024 - 최고의 5개 추천 | 마케팅 가이드"
```

체크:
```
좋음: "SEO 도구 비교 - 최고의 5개 (2024)" (52자)
나쁨: "도구" (너무 짧음, 키워드 부족)
나쁨: "SEO 도구 비교 2024 - 최고의 SEO 도구 무료 SEO 도구 추천" (과도한 키워드)
```

---

**Meta Description (메타 설명)**
- [ ] 길이: 150-160자
- [ ] 주요 키워드 포함 (1회)
- [ ] 행동 유도 (클릭하고 싶게)
- [ ] 고유성 (다른 페이지와 다름)
- [ ] 통계 또는 수치 포함

공식:
```
[문제 상황] + [해결책] + [이점] + [CTA]

예시:
"SEO 도구 선택에 고민이신가요? 2024년 최고의 5가지 도구를
가격, 기능, 사용성으로 비교분석합니다. 당신의 예산과 목표에
맞는 도구를 5분만에 찾아보세요. 지금 비교표를 확인하세요."
```

---

#### B. 콘텐츠 최적화

**키워드 배치**
- [ ] H1 태그: 메인 키워드 1회 (최상단)
- [ ] 도입부 (처음 100단어): 메인 키워드 1회
- [ ] H2 제목: 키워드 변형 2-3회
- [ ] 본문: 자연스럽게 2-3회
- [ ] 결론: 메인 키워드 1회
- **총 밀도**: 1-2% (과도 금지)

측정:
```
키워드 밀도 = (키워드 출현 횟수 / 총 단어 수) × 100

3,000단어 문서:
- 1% 밀도 = 30회 (과도)
- 0.5% 밀도 = 15회 (적절)
- 0.3% 밀도 = 9회 (최소)
```

---

**헤딩 구조 (Heading Hierarchy)**
- [ ] H1: 정확히 1개 (페이지 제목)
- [ ] H2: 3-5개 (주요 섹션)
- [ ] H3: 필요시만 (H2 아래 서브섹션)
- [ ] 순차적 사용 (건너뛰기 금지)
- [ ] 키워드 변형 사용

예시:
```
H1: SEO 도구 비교 2024 - 최고의 5가지
  H2: SEO 도구란? (정의)
  H2: 최고의 SEO 도구 5개
    H3: 도구 1: Ahrefs (설명)
    H3: 도구 2: SEMrush (설명)
  H2: 올바른 SEO 도구 선택 방법
  H2: 무료 vs 유료 SEO 도구 비교
  H2: 결론: 당신에게 맞는 도구 찾기
```

---

**이미지 최적화**
- [ ] Alt 텍스트 작성 (키워드 포함, 자연스럽게)
- [ ] 파일명 최적화 (예: "seo-tools-comparison.jpg")
- [ ] 크기 최적화 (로딩 속도)
- [ ] 해상도: 1200px 이상
- [ ] 관련성: 콘텐츠와 직접 연관

예시:
```
Alt="SEO 도구 5가지 기능 비교표 - Ahrefs, SEMrush, Ubersuggest 포함"
파일명="seo-tools-comparison-2024.jpg"
```

---

**내부 링크 전략**
- [ ] 관련 글로 최소 5-10개 내부 링크
- [ ] 앵커 텍스트: 자연스럽고 키워드 포함
- [ ] 컨텍스트: 링크 전후 문장이 의미 있게
- [ ] 신규/저순위 글 우선 링크
- [ ] 관련도: 링크 대상이 현재 글과 관련

예시:
```
좋음: "자세한 내용은 [SEO 기초 가이드](URL)를 참고하세요."
나쁨: "[클릭](URL)" (앵커텍스트 무의미)
나쁨: "정보가 필요하면 [이 글](URL)을 읽어보세요."
      (앵커텍스트에 키워드 없음)
```

---

**외부 링크 (신뢰도)**
- [ ] 최소 3-5개 신뢰도 높은 외부 링크
- [ ] 소스: 공식 기관, 뉴스사, 학술 자료
- [ ] 컨텍스트: 링크 주변 설명 필요
- [ ] nofollow: 제휴/광고 링크는 rel="nofollow" 필수

신뢰도 등급:
```
AAA: 정부 기관 (한국은행, 통계청), 학술지
AA: 주요 언론 (KBS, 중앙일보), 공개 기관
A: 업계 매체 (마켓인사이트), 전문 블로그
B: 일반 블로그, 커뮤니티
C: 출처 불명확한 사이트
```

---

**콘텐츠 길이**
- [ ] 최소: 2,000 단어 (기본)
- [ ] 권장: 3,000-4,000 단어 (SEO 최적)
- [ ] 상세: 5,000+ 단어 (권위 있는 글)

단어 수에 따른 순위 향상:
```
| 길이 | 순위 향상 | 용도 |
|------|---------|------|
| 1,000-2,000 | 약함 | 최소한 필요함 |
| 2,000-3,000 | 중간 | 기본 권장 |
| 3,000-5,000 | 강함 | 심화 콘텐츠 |
| 5,000+ | 매우 강함 | 권위 수립 |
```

주의: 단순 단어 채우기 금지, 정보 밀도가 중요

### 3단계: 콘텐츠 구조 최적화 (Content Structure Optimization)

#### A. 콘텐츠 아웃라인 템플릿 (Comprehensive)

```

- 문제 상황 제시 (독자의 통증)

- 글의 핵심 미리보기

## CTA (Call-To-Action) - 50-100 단어
- 관련 글 추천

- 댓글 유도
```

#### B. 섹션별 최적화 전략

**도입부 작성 팁**
- 문제로 시작 (독자가 "이게 나야!"라고 느끼게)
- 3-5 문장으로 간단하게
- 핵심 답변 미리 힌트 (계속 읽게 유도)

예시:
```
❌ 나쁜 도입:
"이 글은 SEO 도구에 대해 다룹니다."

✓ 좋은 도입:
"SEO 도구가 너무 많아서 어떤 걸 고르면 좋을지 모르겠나요?
저도 처음에는 그랬어요. 100개 이상의 도구를 시도한 후,
정말 필요한 5가지만 남겨두었습니다.
지금부터 당신에게 딱 맞는 도구를 찾을 수 있습니다."
```

---

**본문 섹션 작성 팁**
- **BLUF** (Bottom Line Up Front): 답변을 먼저 제시
- 세부사항은 나중에
- 3-4 아이디어/포인트 per 섹션
- 각 포인트마다 증거 또는 예시

예시:
```
❌ 역순서:
"SEMrush는... [설명] ...따라서 좋습니다."

✓ BLUF 순서:
"SEMrush는 가장 종합적인 SEO 도구입니다. 왜냐하면:
1) 키워드 리서치
2) 경쟁 분석
3) 백링크 모니터링
모두 지원하기 때문입니다."
```

---

**비교 섹션 작성 팁**
- 표 형식 (한눈에 비교 가능)
- 행: 비교 기준 (가격, 기능, 사용성)
- 열: 각 제품/도구
- 별점 또는 체크마크로 시각화

예시:
```
| 기준 | Ahrefs | SEMrush | Ubersuggest |
|------|--------|---------|-------------|
| 가격 | $$$$ | $$$ | $$ |
| 키워드 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 백링크 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| 경쟁사 분석 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 순위 추적 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
```

### 4단계: 메타태그 작성법 (Meta Tags Best Practices)

#### Meta Description (메타 설명)

구조:
```
[상황/문제] + [해결책 미리보기] + [이점] + [CTA]

예시 분석:
"SEO가 복잡해 보인다면? 이 12가지 체크리스트로

분석:

- 해결: "12가지 체크리스트"
- 이점: "Google Top 10에 올릴 수 있습니다"
- CTA: "지금 무료 가이드를 다운로드하세요"
```

길이 기준:
- 모바일: 120-150자 (Google 모바일 검색 표시 기준)
- 데스크톱: 150-160자
- 최대: 160자 (초과 시 잘림)

### 5단계: 내부링크 전략 (Internal Linking Strategy)

#### A. 내부링크 구조 설계

목표:
- 페이지 간 권위 분산 (Page Authority 전파)
- 사용자 경로 개선
- 크롤링 최적화 (Crawl Efficiency)

구조 유형:

**Silo Structure** (사일로 구조, 권장)
```
Home
├─ SEO (부모)
│  ├─ SEO 기초 (자식)
│  ├─ On-page SEO (자식)
│  └─ Backlink Building (자식)

│  ├─ Content Strategy (자식)
│  └─ Content Calendar (자식)

특징:

- 각 사일로 내 내부 링크 집중
- 카테고리 간 교차 링크 최소화 (단, 관련 있으면 O)
```

#### B. 내부링크 텍스트 (Anchor Text)

규칙:
- [ ] 키워드 풍부 (하지만 자연스럽게)
- [ ] 링크 맥락 명확 (전후 문장 의미 있게)
- [ ] 변형 사용 (같은 앵커텍스트 반복 피하기)
- [ ] 과도한 정확한 매치 피하기 (2-3회만)

예시:
```
❌ 나쁜 예:
"자세한 내용은 [SEO](link)을 참고하세요." (앵커텍스트 너무 짧음)

✓ 좋은 예:
"자세한 내용은 [SEO 최적화 완벽 가이드](link)를 참고하세요."

✓ 매우 좋은 예:
"자세한 내용은 [다음 SEO 가이드](link)에서 다루고 있으니,
혹시 더 알고 싶으면 꼭 읽어보세요."
```

#### C. 링크 배치 전략

**고아 페이지** (고아 페이지: 링크 없는 페이지) 해결
```
문제: 신규 페이지나 저순위 페이지가 발견 안 됨

해결:
1) 높은 권위 페이지에서 링크 (5-10회)
2) 관련 높은 페이지에서 링크 (10회 이상)
3) 컨텍스트 내 링크 (명확한 이유)
```

**Power Pages** 활용
```
전략: 높은 권위 페이지에서 신규 페이지로 링크

예:
- 높은 권위 페이지: "SEO 기초 완벽 가이드" (순위 #1, DA 60)
- 신규 페이지: "On-page SEO 체크리스트" (신규, 순위 X)

→ "SEO 기초" 페이지에서 "On-page SEO"로 링크
→ 권위 전파로 빠른 순위 상승
```

### 6단계: 기술적 SEO 가이드 (Technical SEO Guide)

#### A. 크롤링 최적화 (Crawlability)

**Robots.txt**
```
목적: 검색봇이 접근해야 할 영역과 차단할 영역 지정

기본 예시:
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /private/
Disallow: /test/

Sitemap: https://example.com/sitemap.xml
```

**XML Sitemap**
```
목적: 모든 중요 페이지를 검색봇에게 알리기

구성:
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/article1</loc>
    <lastmod>2024-02-15</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>

주의:
- 자동 생성 도구 사용 (WordPress, Yoast)
- 정기 업데이트 (주 1회)
```

---

#### B. 색인화 (Indexation)

**Canonical Tags** (정규 URL)
```
목적: 중복 콘텐츠 문제 해결

예시:
<link rel="canonical" href="https://example.com/article/">

사용 사례:
- 쿼리 파라미터 (example.com/article?utm_source=...)
- HTTPS vs HTTP
- www vs non-www
```

**Noindex Meta Tags**
```
목적: 특정 페이지를 검색 결과에서 제외

예시:
<meta name="robots" content="noindex, follow">

사용 사례:
- 중복 페이지 (감시 중인 페이지)
- 낮은 품질 페이지
- 임시 페이지 (테스트)
```

---

#### C. Core Web Vitals (핵심 웹 지표)

**LCP** (Largest Contentful Paint)
- 목표: 2.5초 이내
- 개선:
  - 이미지 최적화
  - 불필요한 JS/CSS 제거
  - 캐싱 활용
  - 호스팅 개선 (CDN)

---

**FID** (First Input Delay)
- 목표: 100ms 이내
- 개선:
  - 긴 작업 분할 (long tasks)
  - 타사 스크립트 지연 로딩
  - 메인 스레드 영향 최소화

---

**CLS** (Cumulative Layout Shift)
- 목표: 0.1 이하
- 개선:
  - 이미지/동영상 크기 명시 (width, height)
  - 폰트 로딩 최적화
  - 광고/삽입 공간 예약

## 금지 사항 (Prohibitions)

### 절대 금지 (Zero Tolerance)
1. **Black Hat SEO**: 클로킹, 키워드 스터핑, 비이블 텍스트
2. **링크 구매**: 고급 링크 판매 사이트에서 구매
3. **콘텐츠 자동화**: AI 전체 생성 + 사람 검수 없음
4. **중복 콘텐츠**: 다른 사이트 전문 또는 원본 없이 복사
5. **주제 위배**: E-A-T (전문성) 위배 (의료, 금융 콘텐츠)

### 높은 위험 (High Risk)
6. **과도한 광고**: 광고가 콘텐츠보다 많음
7. **기생충 SEO**: 높은 DA 사이트 내에서 콘텐츠 게시 (이점 착취)
8. **자동 리다이렉트**: 사용자 동의 없이 다른 페이지로 이동
9. **무한 페이지 생성**: 필터로 자동 페이지 생성 (우즈)
10. **잘못된 메타 마크업**: 콘텐츠와 무관한 Schema 사용

### 중간 위험 (Medium Risk)
11. **느린 업데이트**: 3개월 이상 콘텐츠 갱신 없음
12. **죽은 링크**: 404 링크 다수 존재
13. **모바일 비최적화**: 모바일에서 콘텐츠 읽기 어려움
14. **저품질 백링크**: 스팸 사이트의 링크 수락
15. **과도한 키워드**: 키워드 밀도 2-3% 이상
