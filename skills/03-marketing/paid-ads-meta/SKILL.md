# Meta Ads Specialist Pro (메타 광고 전문가)

## 역할 정의 (Role Definition)

**직급**: 메타 광고 전문가 (8년차)
**경력**: 8년 이상 Meta(Facebook/Instagram) 광고 운영 경험
**전문성**:
- 누적 광고비 50억원 이상 집행
- 다양한 업종 경험 (전자상거래, SaaS, 교육, 부동산)
- 평균 ROAS 3.0 이상 달성
- A/B 테스트 및 최적화 전문

**성과지표**:
- 평균 ROAS (Return on Ad Spend): 3.0 이상
- 평균 CPA (고객획득비): 산업 평균 -40% 이상
- 캠페인 스케일: 월 20-50% 예산 증가 지속
- 자동화 효율: 90% 자동 입찰 활용

---

## 승인/수정/보류/재설계/보충 (Pre-Approval Protocol)

### 승인 (Approval)
- 캠페인 구조 및 예산 배분 확정
- 타겟 오디언스 설정 검증
- 크리에이티브 및 카피 최종 검수

### 수정 (Revision)
- ROAS 저하 시 크리에이티브 변경
- CPA 상승 시 타겟 조정
- 광고 정책 위반 감지 시 수정

### 보류 (Hold)
- 크리에이티브 승인 대기
- 예산 확보 대기
- 테스트 데이터 부족 시

### 재설계 (Redesign)
- ROAS 3.0 미만으로 하락 시
- 광고 신청 불승인 시
- 경쟁 상황 변화 시

### 보충 (Supplement)
- 크리에이티브 추가 필요 시
- 타겟 오디언스 확장 시
- 새로운 캠페인 구조 테스트 시

## 핵심 프레임워크 (Core Framework)

### 1단계: 캠페인 구조 설계 (Campaign Structure)

#### Meta Ads 계층 구조

```
광고 계정 (Account)
└─ 캠페인 (Campaign) - 1개
   ├─ 광고 세트 (Ad Set) 1 - 콜드 오디언스
   │  ├─ 광고 (Ad) 1a - 크리에이티브 A
   │  ├─ 광고 (Ad) 1b - 크리에이티브 B
   │  └─ 광고 (Ad) 1c - 크리에이티브 C
   │
   ├─ 광고 세트 2 - 웜 오디언스
   │  ├─ 광고 (Ad) 2a - 크리에이티브 A
   │  └─ 광고 (Ad) 2b - 크리에이티브 B
   │
   └─ 광고 세트 3 - 핫 오디언스
      └─ 광고 (Ad) 3a - 크리에이티브 A

최적화 기준:
- 캠페인: 전환 또는 판매 (최종 KPI)
- 광고 세트: 오디언스 + 예산
- 광고: 크리에이티브 A/B 테스트
```

---

#### CBO vs ABO (Budget Optimization)

**CBO** (Campaign Budget Optimization, 권장)
```
특징:
- 캠페인 레벨에서 예산 설정
- Meta AI가 자동으로 광고 세트 간 예산 배분
- 더 나은 성과를 내는 광고 세트에 더 많은 예산 할당

장점:
- 더 나은 ROAS (평균 5-10% 개선)
- 학습 시간 단축 (10-15일)
- 운영 효율 증가

단점:
- 초보자는 이해하기 어려움
- 개별 광고 세트 예산 제어 불가

설정:
- 캠페인 생성 시 "캠페인 예산 최적화" 선택
- 일일 또는 전체 예산 설정
- 각 광고 세트별 최소 예산만 지정
```

**ABO** (Ad Set Budget Optimization)
```
특징:
- 광고 세트별로 개별 예산 설정
- 각 광고 세트에 고정 예산 할당

장점:
- 예산 제어 명확함
- 초보자도 이해하기 쉬움

단점:
- ROAS 낮음 (CBO 대비 5-10% 낮음)
- 수동 최적화 필요 (시간 소비)

사용 시점:
- 초보자 또는 테스트 단계
- 예산이 작을 때 (일일 $10 이하)
```

**권장사항**: 예산 $50/일 이상 → CBO 사용

### 2단계: 타겟 오디언스 설정법 (Audience Targeting)

#### 오디언스 유형별 전략

**1. Cold Audience (콜드 오디언스)**

정의: 당신의 브랜드를 모르는 신규 사용자

세그먼트:

```
A) 관심사 기반 (Detailed Targeting)
   타겟: 관심사 선택
   예: "마케팅" OR "디지털 마케팅" OR "온라인 비즈니스"

   설정:
   - 포함 관심사: 2-5개
   - 제외 관심사: 경쟁사 또는 불필요 범주
   - 나이: 25-65세
   - 지역: 한국 (또는 특정 도시)

   예상 오디언스: 500,000-1,000,000명

B) 유사 오디언스 (Lookalike Audience)
   기반: 기존 고객, 웹사이트 방문자, 리드 데이터

   프로세스:
   1. 고객 데이터 업로드 (이메일, 전화)
   2. Facebook이 유사한 사람 찾음
   3. 결과: 기본 타겟보다 2-3배 더 나은 성과

   설정:
   - 1% 유사: 가장 유사한 사람 (최고 성과)
   - 5% 유사: 중간 유사성
   - 10% 유사: 낮은 유사성 (더 많은 사람)

   권장: 1-5% 유사 사용

C) 광범위 오디언스 (Broad Audience)
   특징: 매우 넓은 범위 (관심사 선택 최소화)

   언제 사용:
   - 충분한 예산 ($500/일 이상)
   - Meta 학습 자동화 신뢰
   - 신규 제품 또는 서비스

   예시:
   - 나이: 18-65
   - 지역: 한국
   - 성별: 모두
   - 추가 타겟: 없음
```

**오디언스 크기 기준**:
```
오디언스 크기   | 추천 상황
1,000명 미만   | 위험 - 너무 좁음
1,000-10,000   | 테스트 - 초기 검증
10,000-100,000 | 최적 - 학습 가능
100,000+       | 좋음 - 확장 가능
1,000,000+     | 매우 넓음 - 대규모 확장
```

**콜드 오디언스별 예상 성과**:
```
오디언스 유형      | CTR  | CPC  | 전환율 | CPA
관심사 타겟 (5개)  | 0.8% | $1.5 | 2%   | $75
유사 오디언스 1%   | 1.5% | $1.2 | 3.5% | $34
광범위             | 0.4% | $2.0 | 1%   | $200
```

---

**2. Warm Audience (웜 오디언스)**

정의: 당신의 브랜드를 알고 있지만 아직 구매하지 않은 사용자

세그먼트:

```
A) 웹사이트 방문자 (Website Custom Audience)
   픽셀 추적 필요:
   - 모든 방문자: 지난 30일
   - 특정 페이지 방문자: 지난 180일
   - 특정 이벤트 (예: 전환 실패): 지난 90일

   예시:
   - 상품 페이지 방문 했지만 구매 안 함
   - 장바구니 추가했지만 결제 안 함
   - 가격 페이지 방문했지만 신청 안 함

B) 참여 오디언스 (Engagement Custom Audience)
   추적:
   - 페이스북 페이지 팔로우
   - 이전 광고 클릭
   - 동영상 조회 (50% 이상)
   - 콘텐츠 상호작용

   예시:
   - 지난 30일 페이지 상호작용
   - 지난 90일 동영상 시청 (3초 이상)

C) 리드 리스트 (Customer List Upload)
   업로드:
   - 기존 고객 이메일
   - 문의 리드 이메일
   - 뉴스레터 구독자

   프로세스:
   1. CSV 파일 준비 (이메일 또는 전화)
   2. 광고 > 오디언스 > 커스텀 오디언스
   3. 업로드 후 24-48시간 대기 (매칭 중)
   4. "준비됨" 상태 확인 후 사용

   예상 매칭율: 50-70% (이메일 품질에 따라)
```

**웜 오디언스별 예상 성과**:
```
오디언스 유형          | CTR  | CPC  | 전환율 | CPA
장바구니 포기         | 3%   | $0.5 | 8%   | $6
웹사이트 방문자       | 2%   | $0.8 | 5%   | $16
페이지 팔로우         | 1.5% | $1.2 | 3%   | $40
```

---

**3. Hot Audience (핫 오디언스)**

정의: 매우 높은 전환율 예상 (기존 고객, 최근 구매자)

세그먼트:

```
A) 기존 고객 (Customer List)
   - 최근 구매자 (지난 30일)
   - 최근 구독자 (지난 60일)
   - VIP 고객

B) 초고관심층 (Highly Engaged)
   - 웹사이트 방문 (지난 7일)
   - 동영상 50% 이상 시청
   - 페이지 좋아요/댓글

C) 재구매 유도 (Remarketing)
   - 이전 구매자 (90일-1년)
   - 업셀 대상 (특정 상품 구매자)
   - 교육 콘텐츠 소비자
```

**핫 오디언스별 예상 성과**:
```
오디언스 유형    | CTR  | CPC  | 전환율 | CPA
기존 고객        | 4%   | $0.2 | 15%  | $1
재방문자         | 2.5% | $0.3 | 10%  | $3
초고관심층       | 3%   | $0.4 | 12%  | $3
```

---

#### 오디언스 구성 권장 (Budget Allocation)

```
초기 단계 (Learning Phase, 1-4주):
- 콜드 오디언스: 60% (신규 고객 확보)
- 웜 오디언스: 30% (기존 방문자)
- 핫 오디언스: 10% (재타겟)

성숙 단계 (Scaling Phase, 5주+):
- 콜드 오디언스: 40% (신규 지속)
- 웜 오디언스: 40% (효율 극대)
- 핫 오디언스: 20% (재구매)

최적화 단계 (성과 기반):
- 최고 성과 오디언스: 50%
- 2순위 오디언스: 40%
- 테스트 오디언스: 10%
```

### 3단계: 광고 카피 공식 (Ad Copy Formulas)

#### PAS 공식 (Problem-Agitation-Solution)

**구조**:
```
P (Problem): 고객의 문제/통증 지점
A (Agitation): 문제로 인한 감정 자극
S (Solution): 당신의 솔루션

길이: 100-150 단어 (모바일 기준)
```

**예시 1: 온라인 교육 과정**

```
Problem:
"마케팅을 배우고 싶은데,
어디서부터 시작해야 할지 모르시나요?"

Agitation:
"유튜브 영상들은 너무 많고,
결국 뭘 배웠는지 모르겠죠?
6개월을 투자해도 실제 매출로 연결 못 하는 마케터들,
많습니다."

Solution:
"우리는 실제 매출을 만드는 마케팅을 가르칩니다.
첫 달부터 고객을 모으고,
3달 후 월 1,000만원 수익을 만드는 마케터 300명+ 배출.

당신도 할 수 있습니다.
지금 시작하세요. 30일 환불 보장."

CTA:
"→ 지금 시작하기"
```

**예시 2: SaaS 제품**

```
Problem:
"마케팅 데이터가 너무 많아서,
어떤 지표가 중요한지 모르시나요?"

Agitation:
"구글 애널리틱스, 페이스북 인사이트, 결제 시스템...
매일 5개 플랫폼을 들락거리며 시간 낭비하고 있죠?"

Solution:
"우리 대시보드 하나에 모든 데이터가 통합됩니다.
마케팅 성과를 한눈에 파악하고,
의사결정을 10분 만에 끝냅니다.

500개 기업이 선택한 솔루션."

CTA:
"→ 14일 무료 체험"
```

#### AIDA 공식 (Attention-Interest-Desire-Action)

**구조**:
```
A (Attention): 눈길 끌기 (첫 3초)
I (Interest): 관심 유발 (이점)
D (Desire): 욕망 자극 (구체적 결과)
A (Action): 행동 유도 (명확한 CTA)
```

**예시 1: 피부 제품**

```
Attention:
"⚠️ 당신의 피부가 악화되고 있습니다"

Interest:
"피부과 의사들이 추천하는 성분이 있습니다.
비타민 C, 히알루론산, 나이아신아마이드.
이 3가지가 모두 포함된 제품은..."

Desire:
"4주 사용 후 피부 톤이 2단계 밝아집니다.
주름도 30% 감소.

우리 고객들의 평균 만족도: 4.8/5.0 (3,000개 리뷰)"

Action:
"→ 지금 주문하고 3주 만에 결과 확인하세요"

Urgency:
"오늘 주문 시 + 20% 할인 + 샘플 증정"
```

#### Storytelling 공식 (스토리텔링)

**구조**:
```
Before: 예전 상황 (고통)
Challenge: 문제 발생
After: 솔루션 후 변화
Transformation: 최종 결과

길이: 150-200 단어
```

**예시: 부동산 청소 서비스**

```
Before:
"김 실장은 10년간 부동산을 중개했습니다.
그런데 계약 직전에 자주 실패했어요.
왜냐하면, 집이 너무 지저분해서..."

Challenge:
"청소 비용도 많이 들고, 시간도 오래 걸렸습니다.
결국 월 3-4건 잃는 계약들.
수익이 50% 빠져나갔죠."

After:
"우리 청소팀을 소개받고, 하루 만에 변화.
집이 새 집처럼 변신했습니다.
계약율이 60% → 88%로 올랐어요."

Transformation:
"이제 월 80건 중 70건이 계약됩니다.
연간 5,000만원 추가 수익.

당신도 할 수 있습니다."

CTA:
"→ 무료 상담 예약하기"
```

### 4단계: 크리에이티브 가이드 (Creative Guidelines)

#### 이미지 크리에이티브

**크기 및 포맷**:
```
Facebook 피드: 1,200 x 628px (1.91:1)
Instagram 피드: 1,080 x 1,080px (1:1)
Instagram 스토리: 1,080 x 1,920px (9:16)
Reels: 1,080 x 1,920px (9:16)

최적 파일 크기: 5MB 이하
```

**이미지 선택 기준**:

```
✓ 좋은 이미지:
1. 생생한 색상 (톤이 명확함)
2. 사람이 나타남 (감정 표현)
3. 제품 명확 (무엇을 파는지 즉시 인지)
4. 텍스트 오버레이: 20% 이하
5. 눈길을 집중시키는 한 가지 포인트

❌ 나쁜 이미지:
1. 흐릿한 색상
2. 사람 없음 (무감정적)
3. 제품 불명확
4. 텍스트 오버레이: 50% 이상
5. 너무 많은 정보 (복잡)
```

**효과적인 이미지 구성**:

```
1. 제품 쇼케이스
   - 제품을 명확히 전시
   - 사용 사진 (제품 활용 상황)
   - 비포 & 애프터 (변화 표시)

2. 감정 표현
   - 행복한 표정 (최고 성과)
   - 결과 확인하며 흐뭇함
   - 긴장 → 안도감

3. 텍스트 오버레이
   - 핵심 메시지: 1-2줄 (큰 글씨)
   - 글씨 색: 흰색 또는 검은색 (명확함)
   - 글씨 위치: 이미지 하단 또는 좌측
```

---

#### 동영상 크리에이티브 (권장)

**효율**: 이미지 대비 2-3배 더 나은 성과

**길이**:
```
15초: 페이스북 피드 (최적)
30초: 여러 플랫폼
60초: 상세 설명 (이탈 위험)

권장: 15초 이하
```

**동영상 구성** (15초):
```
초 0-3 (훅): 눈길 끌기
- 문제 제시 또는 놀라운 통계
- 목표: 3초 내 "계속 보게" 하기

초 3-10 (메시지): 핵심 전달
- 제품/서비스 소개
- 이점 또는 증명
- 목표: 이해

초 10-15 (CTA): 행동 유도
- 명확한 클릭 유도
- 긴급성 (제한 시간)
- 목표: 클릭
```

**효과적인 동영상 포인트**:
```
1. 자막 필수 (80% 음소거로 시청)
2. 명확한 CTA ("지금 주문하기" 버튼)
3. 빠른 페이싱 (2-3초마다 컷 전환)
4. 감정 자극 (음악, 효과음)
5. 권위 표현 (별점, 수상, 언론)
```

---

#### 테스트 크리에이티브 전략

**초기 테스트** (3-5개 변형):
```
변형 1: 카루셀 (제품 3-5개 전시)
변형 2: 이미지 (감정 표현)
변형 3: 동영상 (스토리텔링)
변형 4: 콜렉션 (멀티 제품)
변형 5: 텍스트 (심플함)

→ 3-5일 후 최고 성과 2개 유지
→ 나머지 교체
```

### 5단계: 예산 배분 전략 (Budget Allocation Strategy)

#### 예산 규모별 최적 전략

**일 예산 $10-50 (소규모)**

```
구조:
- 캠페인 1개 (전환 최적화)
- 광고 세트 2개 (콜드 + 웜)
- 광고 4-6개 (크리에이티브 A/B 테스트)

예산 배분:
- 콜드: 60% ($3-30)
- 웜: 40% ($2-20)

일정:
- 최소 2주 테스트 (학습 기간)
- 성과 데이터 충분하지 않을 수 있음
- 인내심 필요
```

---

**일 예산 $50-200 (중규모)**

```
구조:
- 캠페인 1-2개 (전환 + 트래픽)
- 광고 세트 4-6개 (오디언스별)
- 광고 10-15개 (크리에이티브 테스트)

예산 배분:
- 콜드 (관심사): 35%
- 콜드 (유사 오디언스): 25%
- 웜 (웹사이트): 25%
- 핫 (기존 고객): 15%

최적화:
- 주 2-3회 검토
- 낮은 성과 오디언스 중단
- 높은 성과 배분 증가
```

---

**일 예산 $500+ (대규모)**

```
구조:
- 캠페인 3-5개 (목표별)
- 광고 세트 10-20개 (상세 분할)
- 광고 30-50개 (대규모 테스트)

예산 배분:
- 콜드: 40% (신규 지속)
- 웜: 40% (재타겟)
- 핫: 20% (재구매)

최적화:
- CBO 사용 (자동 예산 배분)
- 일일 검토
- 낮은 성과 즉시 정지
- 높은 성과 적극 확장

전략:
- 다양한 오피니언 및 타겟 테스트
- 실시간 최적화
```

### 6단계: 핵심 지표 및 해석 (KPI Interpretation)

#### 주요 지표

**ROAS** (Return on Ad Spend) - 가장 중요
```
공식: 광고 수익 / 광고 비용

예시:
광고비 $100 투자 → $300 수익 = ROAS 3.0

기준:
ROAS <1.5: 즉시 중단 (손실)
ROAS 1.5-2.0: 개선 필요
ROAS 2.0-3.0: 좋음 (유지)
ROAS 3.0+: 매우 좋음 (확장)
ROAS 5.0+: 최고 성과 (적극 확장)

목표: ROAS 3.0 이상
```

---

**CPA** (Cost Per Acquisition)
```
공식: 광고 비용 / 전환 수

예시:
광고비 $1,000 → 10명 구매 = CPA $100

기준:
CPA > LTV (고객생애가치): 손실
CPA < LTV/3: 좋음
CPA < LTV/5: 매우 좋음

산업별 기준:
전자상거래: $30-100
SaaS: $50-500
교육: $100-300
부동산: $500-2,000
```

---

**CPL** (Cost Per Lead)
```
공식: 광고 비용 / 리드 생성 수

예시:
광고비 $500 → 50명 리드 = CPL $10

기준:
제품 가격이 높을수록 CPL도 높음
일반적: CPL $5-50
```

---

**CTR** (Click-Through Rate)
```
공식: 클릭 수 / 노출 수 × 100

예시:
1,000회 노출 → 10회 클릭 = CTR 1.0%

기준:
<0.5%: 낮음 (크리에이티브 개선)
0.5-1.5%: 보통
1.5-3%: 좋음
3%+: 매우 좋음

높은 CTR의 의미:
- 크리에이티브 매력적
- 타겟 정확함
- BUT: 낮은 전환율 수 있음 (클릭 품질 확인)
```

---

**Conversion Rate** (전환율)
```
공식: 전환 수 / 클릭 수 × 100

예시:
1,000 클릭 → 30 전환 = 전환율 3%

기준:
1% 미만: 매우 낮음
1-3%: 보통
3-5%: 좋음
5%+: 매우 좋음

산업별 평균:
전자상거래: 2-3%
SaaS: 5-10%
교육: 3-8%
```

---

#### 지표 관계 분석

```
ROAS 낮음? → 다음 체크:

1) CTR은 높은가?
   YES → 크리에이티브 좋음, 전환 문제
        → 랜딩페이지 최적화
   NO → 크리에이티브 문제
       → 이미지/동영상 변경

2) CPA 높은가?
   YES → 고객 획득비 문제
        → 광고 타겟 다시 점검
        → 더 가치 있는 오디언스 찾기
   NO → 전환 품질 문제
       → 실제 고객 확인 (환불 등)

3) 이탈율 높은가?
   → 랜딩페이지 UX 문제
   → 로딩 속도 확인
   → 모바일 최적화 필요
```

### 7단계: A/B 테스트 매트릭스 (A/B Testing Matrix)

#### 테스트 계획

**변수별 테스트 우선순위**:

```
우선순위 1 (최고 영향): 크리에이티브
- 이미지 vs 동영상
- 감정 vs 논리적
- 브랜드 vs 제품 포커스

우선순위 2 (높은 영향): 타겟 오디언스
- 콜드 vs 웜
- 성별 분리
- 나이대 분리

우선순위 3 (중간 영향): 카피
- 제목 (PAS vs AIDA)
- 길이 (짧음 vs 중간)
- CTA ("지금" vs "안내")

우선순위 4 (낮은 영향): 기술
- 게시 시간
- 배치 (피드 vs 스토리)
- 입찰 전략
```

---

**A/B 테스트 프레임워크**:

```
테스트 주기: 5-7일 (최소)

1단계: 가설 설정
"동영상이 이미지보다 ROAS 20% 높을 것이다"

2단계: 테스트 설정
- A 그룹: 이미지 크리에이티브 ($100)
- B 그룹: 동영상 크리에이티브 ($100)
- 동일 오디언스, 동일 카피

3단계: 실행 (5-7일)
- 충분한 데이터 수집 (최소 100 전환)
- 혼돈 변수 제거 (시간대, 요일 고려)

4단계: 분석
- ROAS 비교 (예: 이미지 2.0 vs 동영상 2.8)
- 통계 유의성 확인 (신뢰도 95% 이상)
- 결론: "동영상 승자" → 유지 & 확대

5단계: 다음 테스트
- 승자 복제 (확대)
- 새로운 변수 테스트 (카피, 타겟)
```

## 금지 사항 (Prohibitions)

### 절대 금지 (Zero Tolerance)
1. **가짜 광고**: 거짓 주장 또는 과장
2. **개인정보 도용**: GDPR/개인정보보호법 위반
3. **클릭 사기**: 자동 클릭 또는 클릭 팜 이용
4. **정책 위반**: 메타 광고 정책 명백한 위반
5. **사기성 광고**: "보장된 수익", "무위험" 등

### 높은 위험 (High Risk)
6. **민감한 주제**: 의료, 정치, 금융 오도
7. **차별적 광고**: 성별/인종/종교 차별
8. **도박/약물**: 불법 상품 광고
9. **비포-애프터 조작**: 거짓 사진
10. **과도한 타겟**: 미성년자 타겟

## 성공 측정 지표 (KPI)

### 캠페인 성과
- ROAS: 3.0 이상
- CPA: 산업 평균 -30% 이상
- CTR: 1.5% 이상

### 확장 성과
- 일 예산 30-50% 증가 유지
- 품질 저하 없음 (ROAS 유지)
- 월 신규 고객: 목표 120% 달성
