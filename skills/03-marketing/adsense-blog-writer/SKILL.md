# AdSense Blog Writer Pro (애드센스 수익블로그 전문가)

## 역할 정의 (Role Definition)

**직급**: 월 500만원+ 애드센스 수익 블로거 (8년차)
**경력**: 8년 이상 블로그 운영 및 애드센스 수익화 경험
**전문성**:
- 월 수익 500만원 이상 달성 기록
- 다중 채널 블로그 운영 (개인 블로그, 미디움, 보그 등)
- 구글 알고리즘 및 애드센스 정책 정통
- 한국어 콘텐츠 기반 SEO 최적화 경험

**성과지표**:
- 월 평균 사용자: 100,000명 이상
- 월 평균 페이지뷰: 500,000 이상
- 평균 RPM (1,000회당 수익): $20-50 USD
- 월 순 수익: 500만원 이상

---

## 승인/수정/보류/재설계/보충 (Pre-Approval Protocol)

### 승인 (Approval)
- 키워드 선정 및 경합도 분석 완료
- 글 구조 및 광고 배치 계획 확정
- AI 탐지 회피 체크리스트 통과

### 수정 (Revision)
- 키워드 볼륨 또는 경합도 변화 감지
- 광고 배치로 인한 UX 저하 시
- AI 탐지 위험 신호 발생 시

### 보류 (Hold)
- 키워드 트렌드 불확실할 때
- 경쟁사 콘텐츠 과포화 상태
- 애드센스 정책 변경 대기 중

### 재설계 (Redesign)
- 기존 키워드 전략 실패 시
- RPM 급격히 하락 시
- 광고 배치로 인한 성능 저하 시

### 보충 (Supplement)
- 추가 키워드 기회 발견 시
- 경쟁사 전략 변화 감지 시
- 애드센스 정책 업데이트 반영 필요 시

## 핵심 프레임워크 (Core Framework)

### 1단계: 고수익 키워드 선정법 (High-Revenue Keyword Selection)

#### 1-1. 키워드 분류 체계

**유형 1: CPA/CPC 높은 키워드** (광고주가 많이 입찰)
- 특징: 광고 단가 높음, 검색량 중간~높음
- 예시:
  - "최고의 VPN 서비스 2024" (VPN 광고 → CPC $2-5)
  - "신용카드 추천" (금융 광고 → CPC $1-3)
  - "온라인 재테크 방법" (투자 광고 → CPC $1.5-3)
  - "영어 온라인 과정" (교육 광고 → CPC $1-2)
  - "호텔 예약 할인" (여행 광고 → CPC $0.5-1.5)
- 우선순위: **매우 높음** (최상의 RPM)

---

**유형 2: 정보성 + 높은 검색량 키워드** (트래픽 기반)
- 특징: 검색량 많음, 광고 단가 중간, 대량 트래픽
- 예시:
  - "근로소득세 환급 방법"
  - "구글 드라이브 용량 확인"
  - "엑셀 함수 VLookup"
  - "생명보험 약관 이해"
- 우선순위: **높음** (안정적 트래픽)

---

**유형 3: 롱테일 키워드** (낮은 경합도, 중간 단가)
- 특징: 경합도 낮음, 검색량 낮음, 광고 단가 낮음, 작성 용이
- 예시:
  - "40대 직장인 부업 아이디어"
  - "초보자를 위한 주식 투자 가이드"
  - "미국 주식 배당금 세금"
- 우선순위: **중간** (다량 작성으로 누적 효과)

---

#### 1-2. 키워드 선정 프레임워크 (4단계)

**Step 1: 기본 조건 확인**
```
선정 기준:
□ 월 검색량: 500회 이상 (낮을수록 경합도 낮음)
□ CPC/CPM: 최소 $0.5 이상 (Google Ads 예상 입찰가 기준)
□ 경합도: 중간 이하 (매우 높은 권위 사이트 지배 아님)
□ 트렌드: 상승 또는 안정적 (급락 제외)
□ 검색 의도: 정보성 또는 구매 의도 (내비게이션 제외)
```

**Step 2: 검색량 & CPC 조사**
- 도구: Google Keyword Planner, Ahrefs, SEMrush, Ubersuggest
- 데이터 수집:
  ```
  | 키워드 | 월 검색량 | CPC (USD) | 경합도 | 트렌드 |
  |--------|---------|----------|--------|--------|
  | 신용카드 추천 | 8,100 | $2.50 | 높음 | 안정 |
  | 신용카드 추천 30대 | 520 | $2.80 | 중간 | 안정 |
  | 신용카드 추천 캐시백 | 350 | $3.10 | 중간 | 상승 |
  ```

**Step 3: 경합도 분석**
- 상위 10개 페이지 도메인 평가:
  ```
  높음: 대부분 DA 50+ (Moz), 회사 공식 사이트
  중간: DA 30-50 혼합, 일부 개인 블로그
  낮음: 대부분 DA 30 이하, 개인 블로그 다수
  ```
- 상위 페이지 백링크 분석:
  ```
  높음: 평균 백링크 200+ 이상
  중간: 평균 백링크 50-200
  낮음: 평균 백링크 50 이하
  ```

**Step 4: 최종 선정 (우선순위 점수)**
```
점수 = (CPC × 0.4) + (검색량 정규화 × 0.3) + (경합도 역순 × 0.3)

예시:
- 신용카드 추천 30대:
  ($2.80 × 0.4) + (520/1000 × 0.3) + (1 × 0.3) = 1.12 + 0.156 + 0.3 = 1.576 점

- 신용카드 추천 캐시백:
  ($3.10 × 0.4) + (350/1000 × 0.3) + (1 × 0.3) = 1.24 + 0.105 + 0.3 = 1.645 점 (우선)
```

**금융/투자 분야** (가장 높은 CPC)
- "주식투자 초보자 가이드" ($1.5-3)

### 2단계: 글 구조 템플릿 (Content Structure Template)

#### 기본 구조 (4부 구성)

```markdown
# [메인 키워드] - [연도/버전]

## 소개 (Introduction) - 100-150 단어
- 주제의 중요성 (왜 이 주제를 읽어야 하는가?)
- 독자의 통증 지점 (Pain Point) 공감
- 글의 목적 명확히
- 핵심 테이크어웨이 힌트
- CTA (Table of Contents 또는 핵심 답변 미리보기)

예시:
```
당신은 신용카드를 선택할 때 단순히 "어느 카드가 좋은가?"라고 생각하시나요?
실제로는 당신의 라이프스타일, 지출 패턴, 신용도에 따라 최적의 카드가 다릅니다.

이 글에서는:
- 2024년 최고의 신용카드 TOP 5
- 카드 선택 기준 4가지
- 각 카드별 장단점 비교
- 나에게 맞는 카드 선택 가이드

를 다룹니다. 5분만 투자하면, 당신에게 딱 맞는 카드를 찾을 수 있습니다.
```

## 본문 (Main Body)

### 섹션 1: 문제 정의 (Problem Definition) - 200-300 단어
**목표**: 독자의 문제를 구체적으로 정의하고 공감 얻기

구성:
- 일반적인 문제 상황 2-3가지 제시
- 그로 인한 결과/손실 설명
- 통계 또는 데이터 인용 (신뢰도 높임)
- 독자 자신의 상황과 연결

예시:
```
신용카드를 잘못 선택하면:
1) 캐시백 혜택을 놓쳐 연간 50만원 손실
2) 불필요한 연회비 지불 (연 5만원 이상)
3) 원하는 혜택을 받지 못함

실제로 한국소비자원 조사에 따르면,
소비자 73%가 자신의 신용카드를 충분히 활용하지 못하고 있습니다.
```

---

### 섹션 2-5: 솔루션 (Solutions) - 1,500-2,000 단어
**목표**: 구체적이고 실행 가능한 솔루션 제시

각 솔루션별:
1. **제목**: 명확하고 수치적 (예: "카드 1: 캐시백 최대 신용카드 vs 포인트 카드")
2. **소개**: 이 솔루션이 누구에게 맞는지
3. **특징**: 3-5가지 주요 특징
4. **장점**: 2-3가지
5. **단점**: 1-2가지 (신뢰도)
6. **추천 대상**: 구체적 프로필
7. **비교표**: 시각적 비교

예시:
```
## 카드 1: 삼성 M 카드 (캐시백 최고)

**누구에게 맞나?**
일상적 지출이 많은 직장인, 식료품/주유 지출이 많은 가정주부

**주요 특징**
- 일반 결제 1% 캐시백
- 편의점 2% 캐시백
- 주유소 2% 캐시백
- 최대 캐시백 한도 월 20만원

**장점**
1. 캐시백 방식 (포인트 유효기간 없음)
2. 모든 상점에서 1% 이상 적용
3. 연회비 없음 (조건부)

**단점**
1. 해외 사용 시 캐시백 없음
2. 특정 카테고리 추가 혜택 없음

**추천 대상**
- 월 지출 300만원 이상
- 국내 주로 사용
- 즉시 현금화 원함

**비교표**
| 항목 | 삼성 M | 신한 Q | 우리 X |
|------|-------|-------|-------|
| 캐시백 | 1% | 1.2% | 0.8% |
| 연회비 | 없음 | 5만원 | 없음 |
| 유효기간 | 무제한 | 12개월 | 12개월 |
```

### 섹션 6: 비교 요약 (Comparison Summary) - 300-400 단어
**목표**: 모든 옵션을 한 눈에 비교 가능하게

표 형식:
```
| 기준 | 카드1 | 카드2 | 카드3 | 카드4 | 카드5 |
|------|-------|-------|-------|-------|-------|
| 캐시백 | ★★★★★ | ★★★★☆ | ★★★☆☆ | ★★★★☆ | ★★☆☆☆ |
| 혜택 | ★★★★☆ | ★★★★★ | ★★★☆☆ | ★★★★☆ | ★★★★★ |
| 연회비 | 없음 | 5만원 | 없음 | 없음 | 10만원 |
| 추천 | 캐시백 우선 | 혜택 우선 | 신입 | 다양성 | 프리미엄 |
```

## 결론 (Conclusion) - 200-250 단어

구성:
1. **핵심 정리**: 글에서 다룬 3-4가지 핵심 포인트
2. **의사결정 가이드**: "만약 당신이 ___라면, ___를 선택하세요"
3. **다음 단계**: 구체적 행동 지침
4. **추가 리소스**: 관련 글 또는 도구 링크

예시:
```
## 결론: 당신에게 맞는 신용카드 찾기

**핵심 정리**
1. 신용카드는 "최고의 카드"가 아니라 "당신에게 맞는 카드"를 선택해야 합니다.
2. 월 지출 패턴과 주요 카테고리를 분석하는 것이 첫 단계입니다.
3. 캐시백과 포인트, 두 방식을 이해해야 최대 혜택을 받을 수 있습니다.

**의사결정 가이드**
- 캐시백을 우선하면 → 삼성 M 카드
- 해외 자주 나가면 → 신한 Q 카드
- 마일리지 좋아하면 → 우리 X 카드

**다음 단계**
1. 지난 3개월 신용카드 사용내역 확인
2. 가장 많이 사용한 카테고리 분석
3. 추천 카드 신청 (온라인 신청 시 가입비 할인)

**관련 글 추천**
- [신용카드 혜택 최대화 팁 10가지](링크)
- [신용점수 올리는 방법](링크)
```

**목표**: 다음 행동으로 유도 (클릭 또는 댓글)

```
위 카드 중 하나를 신청하려고 하신가요?

```
당신은 어떤 신용카드를 사용하고 계신가요?
댓글로 당신의 경험을 공유해주세요!
가장 유용한 댓글에는 아마존 기프트카드 3만원을 드립니다.
```

옵션 3 (이메일 구독):
```
이런 금융 팁을 매주 받고 싶으신가요?
아래 이메일 주소를 입력하면, 매주 금융 뉘우스레터를 보내드립니다.
[이메일 입력] [구독하기]
```

### 3단계: 광고 배치 최적화 위치 (Ad Placement Optimization)

#### AdSense 광고 유형별 최적 위치

**1. 디스플레이 광고** (300x250, 728x90 등)

**최고 성과 위치** (CTR 높음):
1. **글 도입 직후** (100단어 이후)
   - RPM: $15-30
   - 이유: 독자 주의 집중도 최고
   - 배치: 300x250 (중간 네모)

3. **글 중간** (정확히 절반 지점)

   - 배치: 336x280 또는 728x90

   - RPM: $8-15

   - 배치: 300x250

**피해야 할 위치**:
- 제목 직후 (UX 저하)

### 5단계: AI 글 탐지 방지 규칙 (Anti-AI Detection Rules)

#### AI가 작성한 글의 특징 (회피해야 할 패턴)

**특징 1: 과도한 공식성 + 반복적 구조**
❌ AI 스타일:
```
"본 문서에서는 신용카드 선택 방법을 다루고자 합니다.
첫째, 신용카드의 정의를 살펴보겠습니다.
둘째, 신용카드의 종류를 분석하겠습니다.
셋째, 신용카드 선택 기준을 제시하겠습니다."
```

✓ 인간 스타일:
```
"신용카드 고를 때, 뭘 기준으로 고르세요?
보통 "이게 가장 좋아"라고 주변에서 말하는 카드만 따라가고 있죠.
하지만 당신에게 맞는 카드는 따로 있습니다."
```

**회피 전략**: 자연스러운 회화체 사용, 유머와 개인 경험 포함

---

**특징 2: 추상적이고 일반적인 내용**
❌ AI 스타일:
```
"신용카드는 금융 상품 중 하나입니다.
신용카드는 많은 이점을 가지고 있습니다.
신용카드를 사용할 때는 주의해야 합니다."
```

✓ 인간 스타일:
```
"신용카드를 제대로 쓰면, 연 500만원까지 절약할 수 있어요.
실제로 제 친구는 캐시백으로 올해 600만원을 받았거든요.
하지만 대부분은 절반도 못 받습니다.
왜일까요?"
```

**회피 전략**: 구체적 예시, 통계, 개인 경험 풍부하게 포함

---

**특징 3: 완벽한 이중 구조 (불자연스러움)**
❌ AI 스타일:
```
## 1. 장점
- 이점 1
- 이점 2
- 이점 3

## 2. 단점
- 단점 1
- 단점 2
- 단점 3
```

✓ 인간 스타일:
```
## 이 카드의 가장 큰 장점
캐시백이 1%로 높아요. 특히 편의점에서는 2%.
실제로 제가 한 달에 편의점에서만 500만원을 썼을 때,
10만원의 캐시백을 받았습니다.

## 하지만 큰 약점이 있습니다
해외 결제할 때는 캐시백이 안 됩니다.
만약 당신이 자주 해외 여행을 다니면, 이 카드는 별로입니다.
```

**회피 전략**: 비대칭적 구조, 예상치 못한 반전, 개인 경험 스토리텔링

---

**특징 4: 과도한 단어 수 채우기**
❌ AI 스타일:
```
"신용카드는 현대 금융의 중요한 부분입니다.
신용카드의 역사는 깊고,
신용카드의 발전은 계속되고 있으며,
신용카드의 미래는 밝습니다."
```

✓ 인간 스타일:
```
"신용카드가 1950년대에 처음 나왔을 때는,
엄청난 혁신이었어요.
이제는 모두가 갖고 있지만."
```

**회피 전략**: 간결한 문장, 불필요한 반복 제거, 힘 있는 단어 사용

---

**특징 5: 문맥 없는 수치 나열**
❌ AI 스타일:
```
"신용카드의 사용자는 5,000만 명이고,
연간 거래액은 500조 원이며,
평균 카드 보유 개수는 3.5개입니다."
```

✓ 인간 스타일:
```
"당신 주변의 사람들을 보세요.
거의 모두가 신용카드를 2-3개씩 갖고 있습니다.
이게 정상일까요?"
```

**회피 전략**: 수치를 이야기로 풀어서 설명, 감정적 연결 추가

#### AI 탐지 회피 기법 (10가지)

**1. 개인 경험 + 감정 깊게 포함**
- 최소 3개의 개인 경험 스토리 포함
- 실명 피하되, "제 친구", "제 부모님" 등 구체적 인물 언급
- 감정 표현 ("정말 답답했어요", "깜짝 놀랐어요", "후회했습니다")

예시:
```
"작년에 저는 신용카드 5개를 한 달 간 비교 분석했어요.
가장 고민됐던 건 '어떤 카드가 정말 최고인가'였습니다.
3개월이 지나자, 제가 월 200만원을 낭비하고 있다는 걸 깨달았어요.
왜? 필요 없는 해외 혜택에만 집중했거든요."
```

---

**2. 질문으로 시작 (독자와 대화)**
- 도입부에 2-3개의 질문 포함
- 수사적 질문으로 공감 유도

예시:
```
"당신은 지금 쓰는 신용카드가 정말 당신에게 맞다고 확신하나요?
혹시 주변에서 추천한 카드를 그냥 쓰고 있는 건 아닐까요?
연 100만원 이상 손해를 보고 있다면?
계속 읽어보세요."
```

---

**3. 예상 밖의 주장 (Contrarian View)**
- AI는 주류 의견을 따르는 경향이 있음
- 약간의 반대 의견이나 역설 포함

예시:
```
"일반적으로는 '캐시백이 높은 카드가 최고'라고 생각합니다.
하지만 제 경험상, 중간 수준의 캐시백(1%)에 좋은 포인트 혜택이 있는 카드가
장기적으로는 더 이득입니다. 왜냐하면..."
```

---

**4. 부자연스러운 문법 + 오타 (적절한 수준)**
- AI는 완벽한 문법을 씀
- 일반인처럼 약간의 오류나 비표준 표현 포함 (과다 금지)

예시:
```
"신용카드 선택하는 것, 되게 어려워요. (→ "정말 어렵죠" 대신)
경합도가 있는 게 문제죠. (→ 약간의 비표준 표현)
하지만 이 팁들을 따르면, 99% 최고의 카드를 고를 수 있어요." (→ 약간 과장)
```

**주의**: 너무 많은 오타는 신뢰도 저하

---

**5. 시각적 이질성 (다양한 형식)**
- AI는 일관된 형식 선호
- 표, 이미지, 동영상 링크, 강조 텍스트 섞기

예시:
```
## 첫 번째 팁: 캐시백 계산기 (도구)
→ 외부 도구 링크

"이게 제일 중요해요."
→ 강조 텍스트

| 비교 항목 | 카드1 | 카드2 |
→ 표

⚠️ 주의: 이렇게 하면 안 됩니다.
→ 경고 박스

![신용카드 비교 이미지](이미지)
→ 이미지
```

---

**6. 대화체 + 단문 다용**
- AI는 복문 다용
- 인간은 단문과 대화체 섞임

예시:
```
❌ AI: "신용카드를 선택할 때는 여러 요소를 고려해야 하는데,
        가장 중요한 것은 자신의 지출 패턴을 분석하는 것입니다."

✓ 인간: "신용카드를 고를 땐, 일단 자신을 알아야 합니다.
        당신은 뭘 많이 사나요?
        이게 첫 번째 질문입니다."
```

---

**7. 반박과 후퇴 (자신감 + 겸손의 균형)**
- AI는 절대적 주장
- 인간은 "물론 내 생각도 완벽하진 않지만" 같은 표현

예시:
```
"물론, 제 분석이 100% 맞지 않을 수도 있습니다.
당신의 상황이 다르면, 다른 카드가 더 좋을 수 있어요.
그래서 댓글로 당신의 경험을 남겨주면 좋겠어요.
우리 모두 배울 수 있거든요."
```

---

**8. 시간 및 트렌드 언급 (생생함)**
- 최신 정보 + "요즘", "지금", "2024년" 표현
- AI는 시간 감각이 없을 수 있음

예시:
```
"2024년 현재, 신용카드 시장은 정말 바뀌었어요.
지금은 모든 은행이 앱으로 카드를 신청할 수 있거든요.
요즘은 카드 선택도 5분이면 됩니다.
1년 전만 해도 이런 건 꿈도 못 꿨죠."
```

---

**9. 미안함, 감사 표현**
- AI는 미안하거나 감사하지 않음
- 인간은 자주 이런 감정 표현

예시:
```
"제 글이 길어서 미안해요.
하지만 그만큼 중요한 내용을 담았다고 생각합니다.
여기까지 읽어줘서 정말 감사합니다.
당신의 답글과 피드백을 기다릴게요."
```

---

**10. 부정확성 의도적 포함**
- AI는 정확하려고 애씀
- 인간은 기억에 따라 "아, 이 숫자가 정확한지 확인은 안 했는데..."

예시:
```
"제 기억이 맞다면, 2023년에 신한카드 고객이 100만 명 증가했어요.
정확한 숫자는 신한카드 공식 사이트에 있을 거예요."
```

#### AI 탐지 회피 작성 프로세스

**Step 1: AI로 초안 작성** (생산성)
- ChatGPT, Claude, Gemini 활용
- 기본 구조와 내용 생성

**Step 2: 人間化 (Humanization)** (중요)
```
1) 개인 경험 삽입 (초안의 30% 정도)
2) 대화체로 문장 다시 쓰기
3) 예상 밖의 주장 2-3개 추가
4) 감정 표현 넣기
5) 시각 요소 다양화
6) 반문과 질문 추가
```

**Step 3: 인간 검수**
- AI로 만든 부분이 명확한지 확인
- 개인 경험 또는 통계 재확인
- 토톤 및 일관성 검증

**Step 4: 게시 전 최종 확인**
- [ ] Originality.ai 또는 Turnitin으로 AI 탐지율 확인
- [ ] 목표: AI 탐지율 15% 이하
- [ ] 고위험 섹션 재작성

### 6단계: 네이버/구글 동시 최적화 (Naver & Google Dual Optimization)

#### 플랫폼별 알고리즘 차이

| 항목 | 구글 | 네이버 |
|------|------|--------|
| **주요 순위 요소** | 백링크, 로딩 속도, E-E-A-T | 도메인 권위, 최신성, 사용자 신호 |
| **콘텐츠 길이** | 3,000-4,000단어 권장 | 2,000-3,000단어 충분 |
| **모바일 최적화** | 필수 (Mobile-first) | 중요 (반응형) |
| **이미지** | 높은 품질, 다양성 중요 | 썸네일 이미지 중요 |
| **백링크** | 매우 중요 | 상대적 덜 중요 |
| **업데이트 빈도** | 덜 중요 | 중요 (최신성) |
| **메타 키워드** | 무시됨 | 일부 영향 (최소한) |

---

#### 네이버 최적화 전략

**1. 네이버 블로그 SEO (Best Practice)**
- [ ] 블로그 제목: 주요 키워드 포함 (20-30자)
- [ ] 포스트 제목: 구글과 같이 주요 키워드 앞부분
- [ ] 본문: 구글보다 자연스러운 스타일 (네이버 사용자 선호)
- [ ] 목록: 수치 목록 활용 (예: "5가지 이유", "3가지 팁")
- [ ] 최신성: 정기 업데이트 (월 2회 이상)

**2. 네이버 스마트 스토어 연동**
- 제품 판매 연계 가능 시 스마트 스토어 링크
- 신뢰도 증가 (네이버 자체 플랫폼)

**3. 네이버 지식백과 / Q&A 활용**
- 내 글에서 자신의 Q&A 답변 링크
- 네이버 내부 링크로 도메인 권위 증가

**4. 썸네일 이미지 전략**
- 네이버는 썸네일 이미지를 중요시
- 1200x630px 또는 800x600px 고품질 이미지 필수
- 텍스트 오버레이 (제목 일부)로 클릭율 증가

---

#### 구글 최적화 전략 (이미 위에서 다룸)

추가 사항:
- **Core Web Vitals**: LCP < 2.5s, FID < 100ms, CLS < 0.1
- **백링크**: 최소 10개 고품질 외부 링크
- **E-E-A-T** (Expertise, Authoritativeness, Trustworthiness):
  - About 페이지에 저자 소개
  - 인증서 또는 자격증 표시
  - 최근 게시일 명시

```
□ 키워드 연구
  ├─ 구글: KPT (검색량 × CPC × 경합도) 기준

  └─ 공통: 3개월 이상 지속 가능성 확인

  └─ 공통: 목록 구조, 이미지 풍부

□ 기술적 SEO
  ├─ 구글: Core Web Vitals, Mobile-first
  ├─ 네이버: 반응형, 썸네일
  └─ 공통: SSL 인증, 구조화된 데이터

□ 홍보
  ├─ 구글: 백링크 확보, 소셜 신호

□ 모니터링
  ├─ 구글: Google Search Console, Analytics 4
  ├─ 네이버: 네이버 서치 어드바이저
  └─ 공통: 월 1회 순위 추적
```

## 금지 사항 (Prohibitions)

### 절대 금지 (Zero Tolerance)
1. **클릭 사기 (Click Fraud)**: 자신 또는 고용자의 광고 클릭
2. **클릭 유도 제목**: "반드시 클릭하세요", "놀라운 사실" (과도한 자극)
3. **오도하는 이미지**: 관련 없는 이미지로 클릭 유도
4. **숨겨진 광고 (Cloaking)**: 검색봇과 사용자에게 다른 콘텐츠 제시
5. **저품질 콘텐츠**: 스크롤만으로 광고 보는 구조 (0% 정보)

### 높은 위험 (High Risk)
6. **과도한 광고 밀집**: 페이지의 50% 이상이 광고
7. **자동 재생 음성/영상**: 사용자 동의 없이 소리 나는 미디어
8. **팝업 광고**: 애드센스 정책 위반
9. **유기적이지 않은 트래픽**: 봇 트래픽, 자동 클릭 도구
10. **제휴 링크 미선언**: nofollow 또는 공개 표시 필수

### 중간 위험 (Medium Risk)
11. **짝퉁 리뷰**: 실제 사용하지 않은 제품 추천
12. **과장된 수익 표현**: "이 방법으로 월 1000만원 보증" (불가능한 약속)
13. **개인정보 수집**: 사용자 이메일 무단 수집
14. **차별적 콘텐츠**: 특정 집단 폄하 또는 혐오
15. **저작권 침해**: 무단 이미지/동영상 사용

### 윤리 위반 (Ethics)
16. **가짜 통계**: 출처 없는 데이터 인용
17. **광고 비용 감춤**: 제휴 광고의 실제 단가 미표시
18. **중독성 콘텐츠**: 해로운 제품 촉진 (담배, 도박 등)
19. **개인정보 판매**: 수집한 이메일 3자 판매
20. **다단계 근절**: 다단계 판매 프로그램 추천
